## 問題描述

發展 e-Bike 用 mid-mounted drive unit (coaxial structure)，含 gearbox、motor、drive board、pedaling shaft (torque+angle sensing)。

### 產品目標

- 小型化、輕量化、低噪聲、高輸出扭力

### 規格總覽

| 項目 | 規格 |
|:-----|:-----|
| 最大外徑 | 111 mm |
| 軸向長度 | 92 mm |
| 重量 | ≤ 2500 g |
| Spindle length | < 153 mm |
| 峰值扭矩 (短時 ~3sec) | 125 Nm |
| 連續扭矩 (~8min) | 100 Nm |
| 減速比 | 1:30 ~ 1:35 |
| 馬達最大扭矩 | 5 Nm |
| 最高轉速 | 100 rpm |
| 馬達型式 | SPM 或 IPM |
| 殼體材料 | AZ91D（鎂合金） |
| 齒輪箱材料 | PEEK 或 PA66 |
| 齒輪箱型式 | 多級行星齒輪 / 諧波齒輪 / 擺線齒輪 |

## Step 2: TC 定義 + 候選方向

### Multi-TC 策略：瓶頸路徑

5 個 TC 依賴分析結果：TC1（馬達扭力密度）為部分瓶頸——解決 TC1 後 TC2 自動緩解（同源 SF6），TC4 為 TC1+TC3+TC5 的共同惡化項（殘餘熱），最後處理。

**解題順序**：TC1 → TC3 → TC5 → TC4（殘餘） → TC2（驗證自動緩解）

## Step 3: PC/分離/SF 解法

### TC1: IPM V-type + Halbach 聚磁

**OZ-OT-Px 定義：**
- **OZ** (衝突區): 馬達氣隙 (Ø65mm 內圈)
- **OT** (衝突時): 全速域 (0-100rpm)
- **Px** (衝突參數): 磁通密度

**PC 造句：**
> 馬達磁通密度 Px **必須在氣隙處為高**（為了產生 5Nm 扭矩）**且在徑向尺寸上為低**（為了 OD≤65mm）

**分離策略：** 空間分離 — 氣隙面高磁通 vs 軛部薄型化

**SF 標準解：**
- Class 1.3.2：引入鐵磁物質聚磁（V-type rotor 集中磁路）
- Class 2.2.5：場均勻化（Halbach array 外環消除漏磁，均勻氣隙場）

**解法方案：**
IPM V-type rotor + Halbach array 外環聚磁，NdFeB N48SH 磁石，Ø65mm stator，d-q 向量控制最大化磁阻扭矩。預估 5Nm@Ø65mm 可達成（對比 SPM 同尺寸約 3.2Nm）。

---

### TC3: 低噪齒輪箱

**OZ-OT-Px 定義：**
- **OZ**: 齒輪箱腔體
- **OT**: 嚙合瞬間（每齒進入/退出接觸）
- **Px**: 齒輪嚙合頻率

**PC 造句：**
> 齒輪嚙合頻率 Px **必須為低**（為了降噪至 HPR50 級）**且為高**（為了傳遞 125Nm）

**分離策略：** 整體與部分分離 — 單級大減速比(低嚙合級數) vs 多齒同時嚙合(高承載)

**SF 標準解：**
- Class 1.2.2：引入改性 S2（柔輪/CF-PEEK 彈性齒面吸收衝擊）
- Class 1.2.1：引入 S3 消除有害作用（阻尼材料層消振）

**解法方案：**
- **方案 A**: 諧波齒輪 1:30 單級——柔輪 30%+ 齒同時嚙合→衝擊分散，單級減速→消除多級嚙合噪聲疊加
- **方案 B**: CF-PEEK 斜齒行星齒輪 2 級——碳纖維阻尼自然吸振 + 斜齒漸進嚙合取代直齒瞬時衝擊

---

### TC5: 輕量結構

**OZ-OT-Px 定義：**
- **OZ**: 齒輪齒根 + 殼體鎖付座
- **OT**: 峰值扭矩瞬間 (3sec burst)
- **Px**: 材料密度

**PC 造句：**
> 齒輪/殼體材料密度 Px **必須為低**（為了 ≤2500g）**且機械性能為高**（為了承受 125Nm 峰值衝擊）

**分離策略：** 整體與部分分離 — 基體輕量化 vs 應力集中區局部強化

**SF 標準解：**
- Class 1.1.2：引入 S3 內部添加物（碳纖維作為 PEEK 基體的內部強化）
- Class 5.1.4：利用場的差異（不同應力場區域用不同材料）

**解法方案：**
CF-PEEK (30%CF) 齒輪（σ_flexural≈250MPa vs PA66 的 80MPa）+ 薄壁 AZ91D 殼體（非承載區 2.5mm / 承載區 4mm）+ 不鏽鋼嵌件於鎖付座與軸承座承受反力矩。

---

### TC2: 瓶頸驗證（TC1 從屬）

TC1 解法（IPM Ø65mm, 5Nm）使減速比需求從 1:35 降至 1:25~30 即可達 125Nm。減速比降低 → 齒輪級數/模數可降 → 齒輪箱重量下降約 15-20%。搭配 TC5 的 CF-PEEK 方案，總重預估可控在 2400-2500g。**TC2 自動緩解確認。**

---

### TC4+: PCM 相變熱緩衝（殘餘熱管理）

**OZ-OT-Px 定義：**
- **OZ**: 殼體內壁夾層
- **OT**: 連續工作 8min 週期
- **Px**: 殼體表面積

**PC 造句：**
> 殼體表面積 Px **必須為小**（為了 OD111mm）**且散熱能力為大**（為了 100Nm@8min 溫升 ≤80°C）

**分離策略：** 時間分離 — 峰值熱量用相變緩衝，穩態靠對流散逸

**SF 標準解：**
- Class 5.3.1：相變吸熱（利用潛熱而非顯熱）
- Class 2.2.3：利用多孔/毛細結構（增大 PCM 接觸面積）

**解法方案：**
石蠟 PCM (Tm≈52°C, ΔH≈200kJ/kg) 填充殼體 AZ91D 夾層。
- 估算：連續 100Nm@8min，馬達銅損+鐵損約 50W，8min 熱量 = 50×480 = 24kJ
- 所需 PCM 質量：24kJ ÷ 200kJ/kg ≈ 120g（加 margin → ~129g）
- 佔殼體夾層體積約 150cm³，可容納於 OD111mm 殼壁內
- 休息間隔時自然對流 + 騎行風冷散逸穩態熱，PCM 回復固態
