# TRIZ Step 1: 功能建模

載入 **triz-model** skill，引導建立系統的功能分析模型和質-場診斷。

## 使用方式

```
/triz-model                        # 互動式引導
/triz-model 電池散熱子系統            # 指定子系統
```

## 產出

1. 組件交互圖（有效/有害/不足/過度/缺失）
2. SF 診斷結果（S1, S2, F + 狀態）
3. 改善/惡化自然語言描述 + TC 造句驗證
