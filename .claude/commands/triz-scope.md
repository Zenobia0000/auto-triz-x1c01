# TRIZ Step 0: 問題定向

載入 **triz-scoping** skill，從問題症狀出發，透過結構化根因分析確定子系統和 TC 假設。

## 使用方式

```
/triz-scope                        # 互動式引導
/triz-scope 馬達爬坡時異音            # 帶入症狀描述
```

## 三個子工具

1. **5 Why**（必做）— 快速因果定向
2. **KT Is/Is Not**（有對照組時做）— 問題邊界界定
3. **CECA**（根因仍模糊時）— 完整因果鏈分析
