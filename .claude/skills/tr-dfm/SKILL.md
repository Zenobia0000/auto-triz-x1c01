# TR DFM/DFA 審查

## Overview

本 skill 依製程產出 DFM (Design for Manufacturability) 和 DFA (Design for Assembly) checklist，並執行 BOM 驗證。**不內建任何特定製程或材料** — 從 MC 和 WI 動態識別製程類型並產出對應 checklist。

**宣告：** 「正在使用 tr-dfm skill — 執行 {subsystem} 的 DFM/DFA 審查。」

---

## 輸入

```
/tr-dfm shell              # 指定子系統
/tr-dfm all                # 全部子系統
/tr-dfm assembly           # DFA 審查
/tr-dfm bom                # BOM 驗證
```

| 參數 | 必要 | 說明 |
|:-----|:-----|:-----|
| subsystem / all / assembly / bom | 是 | 審查目標 |

---

## Phase 1: 載入上下文（資料驅動）

1. 從 `.tr-state.json` 讀取子系統清單和對應 WI
2. 讀取目標子系統的 WI 文件
3. 讀取 WI 引用的 MC 文件
4. 讀取相關 ICD 文件
5. 從 MC 文件識別**製程類型**

---

## Phase 2: 製程自動識別

從 MC 和 WI 的內容推斷製程類型，**不硬編碼製程清單**：

| MC/WI 內容特徵 | 製程類型 | 典型 checklist 重點 |
|:--------------|:---------|:-------------------|
| cast/casting/壓鑄/鑄造/die-cast | **壓鑄/鑄造** | 壁厚、拔模角、澆口、縮孔、氣密 |
| injection/molding/射出/注塑 | **射出成形** | 纖維方向(若含纖維)、熔接線、收縮、模溫 |
| machining/CNC/機加工/turning/milling | **CNC 機加工** | 薄壁振動、表面粗度、刀具選型、熱處理 |
| SMC/powder/pressing/粉末/壓制 | **粉末冶金/壓制** | 密度均勻、壓力、脫模、後處理 |
| stamping/沖壓/sheet metal/板金 | **沖壓/板金** | 彎曲半徑、沖孔間距、回彈 |
| forging/鍛造 | **鍛造** | 分模面、飛邊、纖維方向 |
| additive/3D print/積層 | **積層製造** | 支撐結構、表面品質、殘餘應力 |
| PCB/etching/蝕刻 | **PCB 製程** | 線寬/間距、層數、通孔、EMI |
| winding/繞線 | **線圈繞製** | 匝數、絕緣、端部尺寸 |

### 製程通用 checklist 知識（built-in，非產品特定）

以下為各製程類型的**通用工程知識**，用於補充 MC/WI 未涵蓋的 checklist 項目：

**壓鑄/鑄造通用：**
- 壁厚漸變過渡角 ≤ 15°（避免縮孔）
- 流動長度/壁厚比限制（Al: ≤200:1, Mg: ≤150:1, Zn: ≤250:1）
- 密封面平面度需後加工
- 嵌件預熱減少熱應力

**射出成形通用：**
- 收縮率各向異性（含纖維時更顯著）
- 熔接線強度折減（特別是含纖維複合材料：50-70%）
- 模溫影響結晶度（半結晶材料）
- 澆口位置影響纖維配向和熔接線位置

**CNC 機加工通用：**
- 薄壁加工需特殊夾具控制振動
- 難切削材料（Ti/Co/Ni 合金）需低速、高壓冷卻
- 加工硬化風險：避免摩擦、確保正前角
- 粗加工 → 熱處理 → 精加工順序（需應力釋放）

**粉末壓制通用：**
- 高度/直徑比 ≤ 3:1（密度均勻性）
- 最小壁厚 ≥ 2mm
- 底切不可壓制成形（需後加工）
- 蒸氣處理/燒結後處理

## Phase 4: DFA 審查

從 WI-structural（或主整合 WI）和所有 ICD 動態組裝：

```markdown
## DFA Checklist

### 組裝順序（從 WI 步驟推斷）

| 步驟 | 動作 | 工具 | 注意事項 |
|:-----|:-----|:-----|:---------|
| {從 WI 組裝步驟提取} | | | |

### 干涉檢查

| 檢查項 | 方法 | 判定 |
|:-------|:-----|:-----|
| 旋轉件 vs 固定件間隙 | CAD 干涉分析 | 間隙 ≥ 0.5mm |
| 線束/管路路徑 | 3D 佈線模擬 | 不夾不折 |
| 密封壓縮率 | 計算 | 15-25% (O-ring) |

### 緊固規格（從 ICD 提取）

| 位置 | 螺絲 | 扭矩 | 備註 |
|:-----|:-----|:-----|:-----|
| {從 ICD 配合規格表提取} | | | |
```

## Phase 5: BOM 驗證

交叉比對 WI、ICD、MC 和 BOM：

```markdown
## BOM 驗證

| 檢查項 | 方法 | 判定 |
|:-------|:-----|:-----|
| MC 材料 vs BOM 材料 | 比對所有 MC 指定 vs BOM 列出 | 一致/不一致 |
| ICD 介面件 vs BOM | 比對所有 ICD 螺絲/O-ring/膠水 vs BOM | 全含/遺漏 |
| 標準件規格 | 螺絲等級、O-ring 材質 | 正確/錯誤 |
| 替代料記錄 | 原型 BOM vs 量產 BOM 差異 | 已記錄/未記錄 |
```
