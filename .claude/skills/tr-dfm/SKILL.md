# TR DFM/DFA 審查

## Overview

本 skill 依製程產出 DFM (Design for Manufacturability) 和 DFA (Design for Assembly) checklist，涵蓋壓鑄、射出、機加工、SMC 壓制四大製程，並執行 BOM 驗證。

**宣告：** 「正在使用 tr-dfm skill — 執行 {subsystem} 的 DFM/DFA 審查。」

---

## 輸入

```
/tr-dfm shell              # AZ91D 殼體 DFM (壓鑄)
/tr-dfm gearbox            # CF-PEEK 剛輪 DFM (射出) + CoCrMo 柔輪 (機加工)
/tr-dfm motor              # SMC 定子 DFM (粉末壓制)
/tr-dfm all                # 全部子系統
/tr-dfm assembly           # DFA 審查
/tr-dfm bom                # BOM 驗證
```

| 參數 | 必要 | 說明 |
|:-----|:-----|:-----|
| subsystem / all / assembly / bom | 是 | 審查目標 |

---

## 行為

### Phase 1: 載入上下文

1. 讀取目標子系統的 WI（WI-01~04）
2. 讀取對應 ICD（ICD-01~04）
3. 讀取對應 MC（MC-01~06）
4. 讀取 `.tr-state.json` 取得子系統狀態

### Phase 2: DFM Checklist 產出（依製程分支）

### 製程 A: 鎂合金壓鑄 (AZ91D 殼體)

**對應**: WI-04, MC-05, ICD-01/02/04

```markdown
## DFM Checklist — AZ91D 壓鑄殼體

### 壁厚

| 檢查項 | 標準 | 本設計值 | 判定 | 備註 |
|:-------|:-----|:---------|:-----|:-----|
| 最小壁厚 | ≥ 1.5mm (Mg 壓鑄) | 2.5mm | — | |
| 壁厚均勻度 | 最大/最小 ≤ 3:1 | — | — | PCM 腔體區域可能更薄 |
| 壁厚漸變 | 過渡角 ≤ 15° | — | — | 避免縮孔 |

### 拔模角

| 區域 | 標準 | 建議 | 備註 |
|:-----|:-----|:-----|:-----|
| 外壁 | ≥ 1° | 1.5° | |
| 內壁/肋 | ≥ 2° | 2.5° | Mg 收縮率 0.5-0.8% |
| 深腔 (PCM cavity) | ≥ 3° | 3° | 深度/寬度比限制 |
| 嵌件周圍 | N/A (後加工) | — | SUS304 嵌件壓後加工 |

### 澆口與流道

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 澆口位置 | 遠離外觀面和應力集中區 | 建議在分模面附近 |
| 澆口面積 | ≥ 壁厚截面的 30% | |
| 流動長度/壁厚 | ≤ 150:1 (Mg) | |
| 溢料槽 | 最後充填區域需設置 | 排氣用 |

### 孔洞與氣密風險

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 壓鑄氣孔率 | ≤ 1% (X-ray) | PCM 腔體需氣密 |
| 真空壓鑄 | 建議使用 | 降低氣孔率至 0.3% |
| 密封面平面度 | ≤ 0.05mm | ICD-04 O-ring 溝槽 |
| 後加工餘量 | 配合面 +0.5mm | 軸承座、分模面 |

### 嵌件設計

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| SUS304 嵌件固定 | 壓入/螺紋/Loctite | per ICD-01/02 |
| 嵌件預熱 | 150-200°C | 減少熱應力 |
| 嵌件與 Mg 隔離 | PEO 塗層 | per WI-08 防電偶腐蝕 |
| 嵌件拔出力 | ≥ 設計載荷 × 3 | 安全係數 |

### AZ91D 特殊注意

| 項目 | 風險 | 對策 |
|:-----|:-----|:-----|
| 蠕變 | >120°C 時蠕變顯著 | 確認 PCM 隔離效果，殼體不超溫 |
| 腐蝕 | 鎂合金腐蝕敏感 | PEO + 粉體塗裝 per WI-08 |
| 焊接性 | AZ91D 不易焊接 | PCM 密封用機械方式（O-ring/釺焊） |
| 熔湯溫度 | 650-680°C | 模具鋼選 H13，非 P20 |
```

### 製程 B: 射出成形 (CF-PEEK 剛輪)

**對應**: WI-02, MC-04, ICD-02

```markdown
## DFM Checklist — CF-PEEK 30%CF 射出剛輪

### 纖維方向

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 主應力方向 vs 流動方向 | 對齊 | 齒根應力方向必須與纖維方向一致 |
| 纖維方向預測 | Moldflow 分析 | 射出模擬確認齒根區域纖維配向 |
| 性能降級係數 | 垂直方向 ÷ 流向 ≈ 0.5-0.6 | 最弱方向取值 per MC-04 |

### 熔接線

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 熔接線位置 | 不可在齒根 | Moldflow 預測 |
| 熔接線強度 | bulk 的 50-70% | CF 材料尤其嚴重 |
| 澆口數量/位置 | 單澆口優先 | 多澆口→多熔接線 |
| 或使用旋轉芯 | 環形件可考慮 | 消除熔接線 |

### 收縮與翹曲

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 收縮率 | 流向 0.1-0.3%, 垂直 0.5-0.8% | CF-PEEK 各向異性 |
| 模具補償 | 依 Moldflow 收縮預測 | |
| 翹曲量 | ≤ 齒形公差 | 環形件可能碗形翹曲 |
| 保壓曲線 | 需優化 | 厚薄交界處縮痕 |

### 射出參數

| 參數 | 建議範圍 | 備註 |
|:-----|:---------|:-----|
| 模溫 | 180-200°C | PEEK 結晶度控制 |
| 料溫 | 380-400°C | per MC-04 |
| 射速 | 中等 | 過快→噴射；過慢→未填充 |
| 保壓 | 80-120 MPa | 厚壁區域避免縮孔 |

### CF-PEEK 特殊注意

| 項目 | 風險 | 對策 |
|:-----|:-----|:-----|
| 模具磨耗 | CF 極度磨耗模具 | 模仁用粉末冶金鋼或鍍硬鉻 |
| 乾燥 | PEEK 需 150°C/3h 乾燥 | 含水率 <0.02% |
| 結晶度 | 影響強度 30%+ | 模溫 ≥ 180°C 確保結晶 |
| 回收料 | 不建議 for 結構件 | 纖維長度降級 |
```

### 製程 C: CNC 機加工 (CoCrMo 柔輪)

**對應**: WI-02, MC-03

```markdown
## DFM Checklist — CoCrMo F1537 機加工柔輪

### 材料加工性

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 加工難度 | 難切削 (HRC 35-45) | 類似 Inconel |
| 刀具選擇 | 硬質合金 + TiAlN 塗層 | 或 CBN for 精加工 |
| 切削速度 | 30-60 m/min | 低速、高進給 |
| 冷卻液 | 高壓冷卻 (70+ bar) | 強制排屑 |

### 薄壁加工 (柔輪筒壁)

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 最小壁厚 | 設計值 (通常 0.25-0.3mm) | 極薄，需特殊夾具 |
| 振動控制 | 必須 | 薄壁+難切削→振紋 |
| 加工順序 | 先內後外 or 交替 | 控制殘餘應力 |
| 表面粗度 | Ra ≤ 0.8μm (齒面) | 影響疲勞壽命 |

### 齒形加工

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 齒形精度 | JIS 1 級 or AGMA 14 | 諧波齒輪精度要求 |
| 齒根圓角 | ≥ 0.3 × 模數 | 應力集中控制 |
| 線切割 vs 銑削 | 精密銑 or 放電加工 | 齒形複雜度 |
| 去毛邊 | 電化學 or 流體研磨 | 不可手動（損傷風險） |

### 熱處理

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 時機 | 粗加工 → 熱處理 → 精加工 | CoCrMo 固溶+時效 |
| 殘餘應力 | 應力釋放退火 | 精加工前 |
| 硬度 | HRC 35-45 (per MC-03) | 過硬→脆性斷裂 |
| 變形控制 | 熱處理夾具 | 薄壁件易變形 |

### CoCrMo 特殊注意

| 項目 | 風險 | 對策 |
|:-----|:-----|:-----|
| 加工硬化 | 已加工表面硬度上升 | 避免摩擦、確保正前角 |
| 刀具壽命 | 極短（CoCrMo 磨耗嚴重）| 預備充足刀具、即時監控 |
| 粉塵/切屑 | Co 粉塵有毒 | 濕式加工、通風、PPE |
| 成本 | 極高（材料+加工）| 粗胚近淨形（鍛造 or 精密鑄造） |
```

### 製程 D: SMC 壓制 (定子芯)

**對應**: WI-01, MC-02

```markdown
## DFM Checklist — SMC Somaloy 700 5P 壓制定子

### 壓制參數

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 壓制壓力 | 600-800 MPa | per Höganäs 規格 |
| 密度均勻度 | ≤ ±2% | 影響磁性能 |
| 模具間隙 | ≤ 0.02mm | 避免毛邊 |
| 脫模力 | 需計算 | 複雜形狀可能卡模 |

### 幾何限制

| 檢查項 | 標準 | 備註 |
|:-------|:-----|:-----|
| 最小壁厚 | ≥ 2mm | 壓制成形限制 |
| 高度/直徑比 | ≤ 3:1 | 密度均勻性 |
| 齒形 | 可壓制 | SMC 優勢：可做 3D 齒形 |
| 底切 | 不可 | 需後加工 |
| 表面粗度 | Ra 1.6-3.2μm (as-pressed) | 配合面需後研磨 |

### 後處理

| 步驟 | 參數 | 備註 |
|:-----|:-----|:-----|
| 蒸氣處理 | 530°C / 30min | 表面氧化防銹 + 微增強度 |
| 絕緣塗層 | 已含 (5P = 0.5% 磷酸鹽) | 顆粒間絕緣，降低渦流損 |
| 配合面研磨 | Ra ≤ 0.8μm | 氣隙精度 |
| 繞線槽 | 壓製成形 | 不需後加工 |
```

## DFA Checklist — Drive Unit 組裝

### 組裝順序

| 步驟 | 動作 | 工具 | 注意事項 |
|:-----|:-----|:-----|:---------|
| 1 | 下殼裝入主軸承 | 壓入治具 | H7/h6 配合 |
| 2 | 裝入齒輪箱 (剛輪→波發生器→柔輪) | 手動 + 治具 | 柔輪對齊 |
| 3 | 裝入定子 (繞線完成) | Loctite 648 | ICD-01 同心度 |
| 4 | 裝入踏板軸 + 推力軸承 | 壓入 | 軸向預載 |
| 5 | PCM 充填 + 密封 | 注入 + O-ring | ICD-04 洩漏測試 |
| 6 | 上殼合蓋 + 定位銷 | 導銷×2 + M5×6 | ICD-04 |
| 7 | 裝入 PCB + 感測器 | M3 standoff×4 | ICD-03 |
| 8 | 端蓋密封 | FKM O-ring | IP67 |
| 9 | 轉子（上/下）裝入 | 磁吸 — 需治具 | 安全！強磁力 |
| 10 | 最終緊固 | 扭力扳手 | 規定扭矩值 |

### 干涉檢查

| 檢查項 | 方法 | 判定 |
|:-------|:-----|:-----|
| 旋轉件 vs 固定件 | CAD 干涉分析 | 間隙 ≥ 0.5mm |
| 線束路徑 | 3D 佈線模擬 | 不夾不折 |
| O-ring 壓縮率 | 15-25% | per ICD-04 |
| 磁石吸力 vs 治具 | 計算 | 治具保持力 > 磁吸力 × 2 |

### 緊固規格

| 位置 | 螺絲 | 扭矩 | 備註 |
|:-----|:-----|:-----|:-----|
| 殼體合蓋 | M5×6 × n | TBD Nm | Loctite 243 |
| PCB 固定 | M3×8 standoff×4 | 0.5 Nm | ICD-03 |
| 齒輪箱固定 | M4×6 × n | TBD Nm | ICD-02 |
| 端蓋 | M4 × n | TBD Nm | IP67 密封 |

## Phase 4: BOM 驗證

交叉比對 WI、ICD、MC 和 BOM：

```markdown
## BOM 驗證

| 檢查項 | 方法 | 判定 |
|:-------|:-----|:-----|
| MC 材料 vs BOM 材料 | 比對 MC-01~06 指定 vs BOM 列出 | 一致 |
| ICD 介面件 vs BOM | 比對 ICD 螺絲/O-ring/膠水 vs BOM | 全含 |
| 標準件規格 | 螺絲等級 (10.9/12.9)、O-ring 材質 (FKM) | 正確 |
| 替代料 | 原型 BOM vs 量產 BOM 差異 | 已記錄 |
| 成本估算 | 各組件單價 × 數量 | 合理 |
```
