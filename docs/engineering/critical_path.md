## 並行工作建議

### 可完全並行（無依賴）

- WI-01 (AFM) ∥ WI-02 (諧波) ∥ WI-08 (腐蝕) — 指派不同工程師同時啟動
- WI-07 (採購) — 在 T0 同日啟動，與所有 WI 並行

### 可部分並行（有部分依賴）

- WI-03 (PCM) 可在 T0 啟動概念設計，但 T0+2w 才能拿到 WI-01 loss map 做精確模擬
- WI-05 (PCB) 可在 T0 啟動功能定義和元件選型，但 layout 需等 WI-04 包絡線

### 必須序列

- WI-04 必須等 WI-01 + WI-02 + WI-03 的幾何輸出
- WI-06 必須等所有 WI + 物料到位

| 風險 | 影響天數 | 觸發條件 | 緩解 |
