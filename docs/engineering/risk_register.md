# Risk Register — e-Bike Drive Unit v2

> **更新日期**: 2026-04-21
> **來源**: TRIZ Session v2 觀察項 + 工程缺口分析

---

## 風險評級定義

| 等級 | 機率 | 衝擊 |
|:-----|:-----|:-----|
| 高 | >50% 會發生 | 致命：需重大設計變更或專案延遲 >4 週 |
| 中 | 20-50% | 中等：需局部設計調整，延遲 1-3 週 |
| 低 | <20% | 輕微：可在現有設計框架內處理 |

## 風險清單

### R-001: AFM 軸向力不平衡超過推力軸承額定

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 馬達 (AFM) |
| **機率** | 高 |
| **衝擊** | 致命 — 雙轉子磁拉力差若超過推力軸承額定，軸承提前失效 |
| **根因** | AFM 雙轉子結構對磁石位置公差敏感，氣隙不對稱 → 軸向力不平衡 |
| **對策 WI** | WI-01 步驟 7 |
| **緩解措施** | 1. FEA 含磁石位置公差 ±0.1mm 敏感度分析<br>2. 推力軸承選型安全係數 > 2×<br>3. 備選：增加定位銷/磁石夾具提高組裝精度 |
| **驗證** | V3: 軸向力實測 < 推力軸承額定 |
| **狀態** | Open |
| **負責人** | — |

### R-002: CoCrMo 柔輪 125Nm 峰值疲勞失效

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 諧波齒輪 |
| **機率** | 高 |
| **衝擊** | 致命 — 柔輪斷裂 = 齒輪箱完全失效 |
| **根因** | 柔輪每轉承受橢圓交變應力，125Nm 峰值時齒根應力可能接近疲勞極限 |
| **對策 WI** | WI-02 步驟 3 |
| **緩解措施** | 1. 堅持 wrought CoCrMo ASTM F1537（非 cast F75）<br>2. Goodman diagram + S-N 曲線分析，安全係數 > 3<br>3. 齒頂修形降低應力集中<br>4. 表面拋光 Ra ≤ 0.4μm |
| **驗證** | V7: 加速壽命試驗 ≥ 10,000 cycles @ 125Nm/3sec |
| **狀態** | Open |
| **負責人** | — |

### R-003: PCM 密封失效（石蠟滲入馬達腔）

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 熱管理 / 殼體 |
| **機率** | 中 |
| **衝擊** | 致命 — 石蠟接觸繞組或磁石 → 絕緣失效 / 磁路汙染 |
| **根因** | PCM 相變時體積膨脹 ~12%，反覆循環對密封施加交變壓力 |
| **對策 WI** | WI-03 步驟 6, WI-06 Phase C |
| **緩解措施** | 1. 推薦焊接或釺焊密封（非 O-ring/墊片）<br>2. PCM 填充率 85%（預留膨脹空間）<br>3. 獨立小樣先做 100 次循環洩漏測試 |
| **驗證** | V13: 1000 次熱循環後 DSC + 洩漏檢查 |
| **狀態** | Open |
| **負責人** | — |

### R-004: AZ91D 薄壁 (2.5mm) 壓鑄孔洞

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 殼體 |
| **機率** | 中 |
| **衝擊** | 中等 — 孔洞降低局部強度，可能導致 125Nm 峰值時殼體裂紋 |
| **根因** | 鎂合金壓鑄 2.5mm 薄壁難以確保緻密度，氣孔率 > 2% 時強度劣化 |
| **對策 WI** | WI-04 |
| **緩解措施** | 1. 指定真空壓鑄 (vacuum die-cast)<br>2. X-ray 檢驗標準：ASTM E505 Level 2 以上<br>3. 應力集中區壁厚加到 4mm（SUS304 嵌件區域）<br>4. 壁厚敏感度 FEA |
| **驗證** | V11: CMM 尺寸 + X-ray 檢查 |
| **狀態** | Open |
| **負責人** | — |

### R-005: 旋轉磁場 EMI 干擾扭矩感測器 ADC

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 電子 / 感測器 |
| **機率** | 中 |
| **衝擊** | 中等 — SNR < 60dB → 扭矩量測不準 → 助力控制失調 |
| **根因** | AFM 旋轉磁場 + PWM 開關雜訊耦合到感測器信號線 |
| **對策 WI** | WI-05 步驟 5, WI-06 Phase D |
| **緩解措施** | 1. μ-metal 屏蔽片（馬達與 PCB 之間）<br>2. 差動信號走線 + 數位濾波<br>3. 感測器信號 ADC 取樣避開 PWM 開關瞬間<br>4. Phase D 早期做 EMI 量測 |
| **驗證** | V14: ADC SNR ≥ 60dB |
| **狀態** | Open |
| **負責人** | — |

### R-006: SMC 定子鐵損 @高頻超出 20W 預算

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 馬達 (AFM) |
| **機率** | 中 |
| **衝擊** | 中等 — 鐵損超標 → 效率下降 + PCM 熱負荷增加 |
| **根因** | SMC 在 >500Hz 時渦流損和磁滯損非線性增加，8 極 AFM @3000rpm 電頻率 = 200Hz（基波），但諧波分量可能延伸到 kHz |
| **對策 WI** | WI-01 步驟 5 |
| **緩解措施** | 1. FEA 頻率掃描：100Hz~5kHz 損耗曲線<br>2. 比較 Somaloy 700 5P vs 700HR vs 500（不同損耗特性）<br>3. 若超標 → 降低極數到 6 極（電頻率降 25%）或加大定子截面 |
| **驗證** | V4: 量熱法實測核心損耗 ≤ 20W @3000rpm |
| **狀態** | Open |
| **負責人** | — |

### R-007: AZ91D / SUS304 電偶腐蝕

| 項目 | 內容 |
|:-----|:-----|
| **子系統** | 殼體 / 結構 |
| **機率** | 中 |
| **衝擊** | 中等 — 鎂合金在濕環境下快速腐蝕，影響結構壽命 |
| **根因** | 電偶電位差 ΔE ≈ 1.5V（AZ91D: -1.6V SCE, SUS304: -0.1V SCE），屬嚴重電偶腐蝕風險 |
| **對策 WI** | WI-08 全文 |
| **緩解措施** | 1. AZ91D 表面 PEO/陽極氧化 ≥ 50μm<br>2. SUS304 嵌件與 AZ91D 間 PEEK 墊圈隔離<br>3. 環氧結構膠黏合 + 密封<br>4. 成品 IP67 等級密封 |
| **驗證** | V12: ASTM B117 鹽霧 500hr, 腐蝕深度 ≤ 0.05mm |
| **狀態** | Open |
| **負責人** | — |
