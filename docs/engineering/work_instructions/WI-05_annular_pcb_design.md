## Step 2: 關鍵元件選型

### 2.1 功率級

| 元件 | 推薦 | 理由 |
|:-----|:-----|:-----|
| MOSFET | GaN (EPC2302 or equiv.) | 低 Rds_on、低寄生電感、高頻開關 |
| 備選 | Si MOSFET (IRFH5015) | 成本低、48V 充裕 |
| Gate Driver | 三相整合 IC (DRV8323 or equiv.) | 縮小面積 |
| 電流感測 | 低側 shunt × 3 (1-2mΩ) | 成本低、精確 |

### 2.2 MCU

| 推薦 | 特點 |
|:-----|:-----|
| **ST STM32G4** | 硬體 FOC 加速器 (CORDIC+FMAC)、170MHz、12-bit ADC ×5 |
| TI TMS320F280x | DSP 核心、成熟 FOC 庫 |

### 2.3 通訊

| 介面 | 用途 |
|:-----|:-----|
| CAN 2.0 / CAN-FD | 與車架控制器/顯示器通訊 |
| UART | debug / 韌體更新 |

## Step 3: 環形 Layout 設計

### 3.1 區域規劃

```
環形 PCB 俯視圖 (展開為角度分區):

  0°────90°───180°───270°───360°
  │  Power  │ Gate  │  MCU  │ Sensor │
  │  Stage  │Driver │       │  ADC   │
  │ U/V/W   │       │ CAN   │ Torque │
  │ DC bus  │       │ UART  │ Pos.   │
  │ Cap     │       │       │        │
```

### 3.2 Layout 關鍵規則

| 規則 | 理由 |
|:-----|:-----|
| 功率級和感測器 ADC 分區 180° 對角 | 最大化 EMI 隔離 |
| DC bus 去耦電容緊鄰 MOSFET drain | 最小化迴路電感 |
| 相電流 shunt 對稱放置 | 均勻電流量測 |
| 地平面不分割（統一 GND） | 避免地迴路 |
| 扭矩感測器走差動線 | 抗共模雜訊 |
| 高度限制: 底面元件 ≤ 3mm, 頂面 ≤ 5mm | 軸向 8mm 預算 |

### 3.3 散熱設計

1. MOSFET 背面 thermal vias → PCB 底面 copper pad
2. 底面 copper pad → 導熱矽膠墊 → 端蓋 (AZ91D, k=72 W/mK)
3. 熱阻目標：Junction-to-case ≤ 5°C/W

## Step 4: EMI 防護（風險 R-005）

### 4.1 EMI 來源

| 來源 | 頻率 | 耦合路徑 |
|:-----|:-----|:---------|
| AFM 旋轉磁場 | f_elec = 4×n/60 (max ~200Hz 基波) | 磁場直接耦合到 PCB |
| PWM 開關雜訊 | f_sw = 20-40kHz + 諧波 | 傳導 (gate/drain loop) |
| 諧波嚙合振動 | f_mesh ≈ 2.9kHz | 機械振動 → PCB microphonics |

### 4.2 防護措施

| 措施 | 實施 | WI 步驟 |
|:-----|:-----|:--------|
| **μ-metal 屏蔽片** | 馬達端面與 PCB 之間放置 0.5mm μ-metal 片 | Layout |
| 差動信號 | 扭矩感測器信號走差動對 + 共模扼流圈 | 原理圖 |
| ADC 取樣時機 | 避開 PWM 開關瞬間（死區取樣）| 韌體 |
| 接地策略 | 統一 GND，但功率迴路和信號迴路走線不重疊 | Layout |
| 輸入濾波 | 感測器 ADC 前置 RC 低通 (fc ≈ 1kHz) | 原理圖 |

### 4.3 驗證

V14: ADC SNR ≥ 60dB，在馬達全速運轉 + 滿載條件下量測

## Step 5: 韌體控制規格

### 5.1 FOC 控制架構

```
Torque command (from CAN)
      │
      ▼
  MTPA lookup → Id_ref, Iq_ref
      │
      ├──→ PI current loop (Id) → Vd
      │
      └──→ PI current loop (Iq) → Vq
                                    │
                              dq→αβ→SVM → PWM output
```

### 5.2 關鍵參數

| 參數 | 值 | 來源 |
|:-----|:---|:-----|
| PWM 頻率 | 20 kHz (初始) | 平衡效率和噪聲 |
| 電流環帶寬 | ≥ 2 kHz | 響應速度 |
| 位置回授 | Sensorless (back-EMF observer) 或 Hall | 依需求 |
| MTPA 角度 | WI-01 FEA 結果 | 依 IPM 特性 |
| 場弱轉速 | > base speed (WI-01 定義) | — |
| 過流保護 | 峰值電流 × 1.2 | 安全 |
| 過溫保護 | NTC @殼體, T_limit = 80°C | 降功率 |

### 5.3 踏板助力演算法

| 參數 | 描述 |
|:-----|:-----|
| 輸入 | 踏板扭矩 (Nm), 踏板角速度 (rpm), 車速 (km/h from CAN) |
| 輸出 | 馬達扭矩指令 (Nm) |
| 助力比 | 1:1 ~ 1:4 (依 assist level) |
| 延遲 | ≤ 50ms（騎乘感覺自然） |
| 法規 | EU: 限速 25 km/h 斷電 |
