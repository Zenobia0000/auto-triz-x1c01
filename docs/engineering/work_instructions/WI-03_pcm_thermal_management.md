## Step 1: 熱負荷分析

### 1.1 熱源識別

| 熱源 | 位置 | 功率(W) | 來源 |
|:-----|:-----|:--------|:-----|
| 馬達銅損 | 定子繞組 | WI-01 loss map | 主要 |
| 馬達鐵損 | 定子芯 | WI-01 loss map | 次要 |
| 齒輪箱摩擦熱 | 齒面+軸承 | WI-02 效率計算 | 主要 |
| 驅動板損耗 | PCB FET | ~10W 估算 | 次要 |
| **合計** | | **~90-120W** (精確值待 WI-01/02) | |

### 1.2 初步估算（WI-01 loss map 未到前可先用）

```
假設連續工況總損耗 Q_total ≈ 90W (保守)
8min 總熱量: E = 90W × 480s = 43,200 J = 43.2 kJ
```

---

## Step 2: PCM 設計計算

### 2.1 PCM 需求量

```
PCM 需吸收的熱量 = Q_total × t_continuous - Q_dissipated × t_continuous

假設工作期間殼體對流散熱 Q_dissipated ≈ 30W (低風速)
→ PCM 需吸收: (90-30) × 480 = 28,800 J = 28.8 kJ

PCM 質量 = E_absorb / ΔH = 28,800 / 170,000 = 0.169 kg ≈ 170g

安全裕量 1.5×: 170 × 1.5 = 255g ≈ 250g (TRIZ spec)
```

### 2.2 PCM 體積

```
RT55 液態密度 ≈ 0.77 kg/L
體積 = 250g / 0.77 = 325 mL

膨脹裕量 (固→液 ~12%): 腔體體積 = 325 / 0.85 = 382 mL ≈ 400 mL
```

### 2.3 腔體幾何

```
殼體夾層環形腔體:
  外徑 = 殼體 ID + 2×腔體厚度 ≤ 殼體 OD - 2×外壁厚度
  
  可用環形截面積 A_cavity = π/4 × (D_outer² - D_inner²)
  腔體長度 L_cavity = 體積 / A_cavity
  
  例: 齒輪箱段 (壁厚充裕)
    D_outer = 105mm (殼體 OD111 - 外壁 3mm×2)
    D_inner = 95mm (殼體 ID ~92 + 內壁 1.5mm×2)
    A = π/4 × (105² - 95²) = π/4 × 2000 = 1571 mm²
    L = 400,000 / 1571 = 255 mm → 不可能(殼體只有 92mm 長)
    
  需多段腔體或增加腔體厚度:
    調整 D_inner = 92mm, D_outer = 108mm (腔體厚 8mm per side)
    A = π/4 × (108² - 92²) = π/4 × 3200 = 2513 mm²
    L = 400,000 / 2513 = 159mm → 仍超
    
  結論: PCM 腔體需分佈在齒輪箱段+馬達段才夠體積
```

## Step 3: 暫態熱模擬

### 3.1 軟體選擇

| 軟體 | 適用性 | 建議 |
|:-----|:-------|:-----|
| ANSYS Fluent | PCM 相變模擬（Solidification/Melting model）| **首選** |
| COMSOL Multiphysics | 多物理場耦合 | 備選 |
| ANSYS Mechanical (Thermal) | 簡化固體導熱 | 快速初估 |

### 3.2 模型建立

1. 建立 2D 軸對稱模型（簡化計算，後期 3D 驗證）
2. 幾何：殼體截面（外壁 + PCM 腔 + 內壁 + 氣隙 + 馬達/齒輪箱）
3. 材料卡：

| 區域 | 材料 | k (W/mK) | ρ (kg/m³) | Cp (J/kgK) | 特殊 |
|:-----|:-----|:---------|:----------|:-----------|:-----|
| 殼體 | AZ91D | 72 | 1810 | 1020 | — |
| PCM | RT55 | 0.2 | 770(液)/880(固) | **焓-溫度曲線** | 相變模型 |
| 定子 | Somaloy | 20 (3D isotropy) | 7350 | 450 | — |
| 繞組 | 銅(等效) | 1.5 (含絕緣) | 5500(等效) | 400 | — |
| 空氣 | Air | 0.026 | 1.18 | 1006 | — |

### 3.3 邊界條件

| 邊界 | 類型 | 值 | 工況 |
|:-----|:-----|:---|:-----|
| 外壁 (靜止) | 自然對流 | h = 10 W/m²K | 停車 (worst case) |
| 外壁 (騎乘) | 強制對流 | h = 25 W/m²K | 15 km/h |
| 外壁 (高速) | 強制對流 | h = 50 W/m²K | 30 km/h |
| 內壁 (馬達面) | 熱源 | Q = loss map | WI-01 |
| 內壁 (齒輪面) | 熱源 | Q = 齒輪損耗 | WI-02 |
| 踏板軸 | 絕熱 | q = 0 | 保守假設 |
| 環境 | 溫度 | 25°C / 40°C | 標準 / 最惡劣 |

### 3.4 PCM 相變設定

**關鍵**: PCM **不可用恆定 Cp** 建模！必須使用焓-溫度曲線：

```
焓 H(T) 曲線 (RT55):
  T < 45°C:  H = Cp_solid × T = 2.0 kJ/kgK × T
  45 ≤ T ≤ 57°C: H = H_45 + ΔH × (T-45)/12  (線性近似相變區)
  T > 57°C:  H = H_57 + Cp_liquid × (T-57) = H_57 + 2.0 × (T-57)
```

在 Fluent 中：啟用 Solidification/Melting model，設定 Solidus=45°C, Liquidus=57°C, Latent Heat=170 kJ/kg

### 3.5 網格要求

| 區域 | 最小網格尺寸 | 層數 |
|:-----|:------------|:-----|
| **PCM 腔體** | ≤ 1mm | **≥ 5 層** (穿過厚度方向) |
| 殼體壁 | 1mm | 3 層 |
| 馬達/齒輪（簡化塊）| 2mm | — |

### 3.6 求解設定

| 工況 | 初始溫度 | 熱源 | 對流 | 時間 | 時間步 |
|:-----|:---------|:-----|:-----|:-----|:-------|

| 問題 | 對策 |

## Step 5: 穩態散熱驗證

### 5.1 穩態散熱能力

```
Q_dissipated = h × A_surface × (T_surface - T_ambient)

殼體表面積 A ≈ π × 111mm × 80mm ≈ 27,900 mm² = 0.028 m²
加鰭片 ×1.5 倍: A_eff ≈ 0.042 m²

騎乘 (h=25): Q = 25 × 0.042 × (80-25) = 57.8W
高速 (h=50): Q = 50 × 0.042 × (80-25) = 115.5W
```

**結論**: 純對流散熱 58-116W，連續 90W 在高速時可維持穩態，低速時需 PCM 緩衝。

## Step 6: 密封設計（風險 R-003）

> **安全關鍵項 — PCM 洩漏可能導致馬達失效**

### 6.1 密封方式選擇

| 方式 | 可靠性 | 成本 | 推薦 |
|:-----|:-------|:-----|:-----|
| **焊接密封** (雷射焊/摩擦焊) | 最高 | 高 | **首選** |
| 釺焊密封 | 高 | 中 | 備選 |
| O-ring + 螺栓壓緊 | 中（循環後可能劣化） | 低 | **不推薦** |
| 環氧灌封 | 中 | 低 | 小量原型 |

### 6.2 密封設計要點

1. PCM 注入口位於殼體上方（液態注入後密封）
2. 排氣口位於最高點（注入時排出空氣）
3. 注入完成後：雷射點焊密封注入口和排氣口
4. 密封焊道寬度 ≥ 1mm、深度 ≥ 壁厚 50%

### 6.3 洩漏測試

1. 充氣加壓 0.2 MPa，保持 10 min
2. 判定：壓降 ≤ 0.01 MPa
3. 納入 WI-06 Phase C 測試項目
