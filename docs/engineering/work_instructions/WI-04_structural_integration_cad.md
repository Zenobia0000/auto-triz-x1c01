# WI-04: 結構整合與 CAD 組裝程序

> **版本**: 1.0 | **日期**: 2026-04-21
> **負責域**: 結構 / 機械設計
> **前置條件**: WI-01 (馬達幾何+軸向力), WI-02 (齒輪箱幾何), WI-03 (PCM 腔體厚度), WI-08 (嵌件隔離規格)

## Step 1: 座標系與基準建立

### 1.1 建立設計座標系

1. 開啟 CAD 軟體（SolidWorks / NX / CATIA）
2. 定義全域座標系：
   - **原點**: 踏板軸中心線與 drive-side 端蓋配合面的交點
   - **Z 軸**: 踏板軸軸線方向（正方向 = non-drive side）
   - **X 軸**: 垂直向上
   - **Y 軸**: 右手定則
3. 定義基準面：
   - **Datum A**: 踏板軸中心軸線（所有旋轉件的基準）
   - **Datum B**: Drive-side 端蓋配合面（軸向基準 Z=0）
   - **Datum C**: 定位銷孔（角度基準）

### 1.2 包絡線約束

建立以下不可侵犯的包絡線（作為參考幾何，不可刪除）：

```
外徑包絡: Ø111mm 圓柱（含安裝法蘭）
軸向包絡: Z=0 ~ Z=92mm
踏板軸包絡: Ø30mm × L153mm（中心通道）
```

## Step 2: 子組件建模（Level 0 → Level 4）

### 2.0 Level 0: 踏板軸包絡線

**這是不動的基準 — 所有設計圍繞它展開。**

1. 建立踏板軸中心通道：Ø30mm，長度 153mm
2. 標註踏板軸支撐軸承位置（左右兩端）
3. 標註扭矩感測器安裝區域（Z 位置待 Level 3 確定）
4. 標註輸入端（rider side）和輸出端（chainring side）

```
┌─────────────────── 153mm ───────────────────┐
│  rider side  │ sensor │ motor │ gear │ output │
│              │  5mm   │ 18mm  │ 28mm │        │
└──── Ø30mm ─────────────────────────────────── ┘
                        ↑ Datum A (軸中心)
```

### 2.1 Level 1: 馬達 + 齒輪箱

**馬達和齒輪箱的外形決定殼體的 ID — 必須先建模。**

#### 2.1.1 AFM 馬達子組件

1. 匯入 WI-01 的馬達 3D 幾何 (D8)
2. 如無 3D，依以下參數建立：

| 零件 | 外徑 | 內徑 | 軸向厚度 |
|:-----|:-----|:-----|:---------|
| 上轉子盤 | Ø100mm | Ø32mm | 3mm (back iron) + 3mm (magnet) |
| 氣隙 (上) | — | — | 0.8mm |
| 定子 (SMC) | Ø100mm | Ø32mm | ~8mm (含齒+軛) |
| 氣隙 (下) | — | — | 0.8mm |
| 下轉子盤 | Ø100mm | Ø32mm | 3mm (magnet) + 3mm (back iron) |
| **合計** | | | **~18mm** (含公差裕量) |

3. 將馬達子組件放置在軸向位置：
   - 以 Datum B 為基準，馬達位於 Z = Z_motor_start
   - Z_motor_start 由殼體結構決定（Level 2 回頭調整）

#### 2.1.2 諧波齒輪箱子組件

1. 匯入 WI-02 的齒輪箱幾何
2. 如無 3D，依以下參數建立：

| 零件 | 外徑 | 內徑 | 軸向厚度 |
|:-----|:-----|:-----|:---------|
| 圓形齒輪 (circular spline) | Ø90mm | — | ~10mm |
| 柔輪 (flex spline) | Ø~88mm | — | ~15mm (含杯形) |
| 波發生器 (wave generator) | Ø~60mm | Ø30mm | ~8mm |
| **合計** | | | **~28mm** |

3. 放置在馬達之後：Z = Z_motor_end → Z_motor_end + 28mm

#### 2.1.3 軸向堆疊驗證

```
Z=0 ── 端蓋 A (drive side)
       │ 齒輪箱: 28mm
       │ 馬達: 18mm
       │ PCB: 8mm
       │ 感測器: 5mm
       │ 端蓋 B (non-drive side)
Z=92 ─ 包絡線上限

檢查: 28+18+8+5+端蓋×2 = 59mm + 端蓋
端蓋厚度預算: (92-59)/2 = 16.5mm each (充裕)
```

**注意**：組件軸向位置在此步驟為初步放置，Level 2 完成後可能微調。

### 2.2 Level 2: 殼體 + 軸承佈局

#### 2.2.1 殼體 ID/OD 計算

```
殼體 ID = 馬達 OD + 間隙 × 2
        = 100mm + 1mm × 2 = 102mm (馬達段)

殼體 OD = 111mm (固定約束)

壁厚 = (111 - 102) / 2 = 4.5mm (馬達段)

但: TRIZ spec 要求壁厚 2.5-4mm
    → 調整間隙或接受局部 4.5mm
    → 齒輪箱段: OD90 + 間隙 → 壁厚更厚 → 容納 PCM 腔體
```

1. 建立殼體基本形狀：
   - 外輪廓：Ø111mm 圓柱
   - 內輪廓：階梯式（馬達段 ID102、齒輪箱段 ID92、PCB 段 ID 依需求）
   - 上下分割面：選擇在馬達/齒輪箱分界處（方便組裝）

#### 2.2.2 殼體分割策略

```
上蓋 (Non-drive side):
  ├── 包含: 感測器安裝區 + PCB 安裝區
  ├── 材料: AZ91D
  └── 連接: 螺栓 + O-ring

下蓋 (Drive side):
  ├── 包含: 齒輪箱腔 + 馬達腔 + PCM 腔體
  ├── 材料: AZ91D
  └── 連接: 車架鎖付座（SUS304 嵌件）
```

#### 2.2.3 軸承佈局

1. 取 WI-01 推力軸承需求 (D7)
2. 配置軸承：

| 位置 | 軸承類型 | 承受載荷 | 建議型號系列 |
|:-----|:---------|:---------|:------------|
| 輸入側 (rider) | 深溝滾珠 | 徑向 | 6x05 系列 |
| 輸出側 (chain) | 深溝滾珠 | 徑向 + 輸出扭矩反力 | 6x06 系列 |
| 馬達推力 | 推力滾珠 or 角接觸 | 軸向 (AFM 磁拉力) | 51100 系列 or 整合角接觸 |
| 波發生器 | 薄截面交叉滾子 | 諧波內部 | WI-02 指定 |

3. 在 CAD 中放置軸承，確認：
   - 軸承座孔直徑和公差 (H7 for 外圈)
   - 軸承座軸向定位面
   - 潤滑脂空間

#### 2.2.4 SUS304 嵌件設計

1. 識別嵌件位置：
   - 軸承座（承受軸向+徑向力）
   - 車架鎖付座（承受反扭矩 125Nm）
   - M5/M6 螺栓孔（殼體上下蓋鎖固）
2. 嵌件固定方式選擇：

| 方式 | 優點 | 缺點 | 推薦 |
|:-----|:-----|:-----|:-----|
| 壓鑄嵌入 (cast-in) | 強度最高、無後加工 | 模具複雜、嵌件需預處理 | 軸承座 |
| 螺紋嵌件 (threaded insert) | 後加工安裝、可更換 | 強度較低 | 螺栓孔 |
| 壓配 (press-fit) | 中等強度 | 需精密公差 | 鎖付座 |

3. **電偶隔離**（參照 WI-08）：
   - 所有 SUS304 與 AZ91D 接觸面需 PEEK 墊圈或環氧膠隔離
   - 壓鑄嵌入時：嵌件表面先噴 PEO 陶瓷層

### 2.3 Level 3: PCM 腔體 + PCB + 感測器

#### 2.3.1 PCM 腔體

1. 取 WI-03 的腔體厚度 (3-5mm)
2. 在殼體壁厚中配置夾層：

```
截面示意 (徑向):

│← 壁厚 →│
│ 外壁 │ PCM │ 內壁 │
│ 1.5mm│ 4mm │ 1.5mm│  = 7mm total (齒輪箱段，壁厚充裕)
│ 1.5mm│ 3mm │ 1.0mm│  = 5.5mm total (馬達段，空間較緊)
```

3. 設計 PCM 注入口和排氣口（位於殼體上方，組裝後密封焊接）
4. 預留膨脹空間：腔體體積 = PCM 體積 / 0.85

#### 2.3.2 環形 PCB

1. 定義 PCB 包絡線（交付 WI-05）：
   - 外徑：≤ 殼體 ID（該段）- 2mm 間隙
   - 內徑：≥ 踏板軸 OD + 2mm 間隙
   - 軸向：8mm（含元件高度）
   - 安裝方式：端蓋上的銅柱 standoff（3-4 個，等角分佈）
2. 建立 PCB 佔位塊（placeholder）
3. 標註連接器開口位置（線束出口）

#### 2.3.3 感測器

1. 磁致伸縮扭矩感測器安裝位置：
   - 安裝在踏板軸上，軸向 5mm
   - 位於 PCB 和 non-drive 端蓋之間
   - 信號線穿過端蓋到 PCB
2. 確認感測器 OD 不干涉殼體 ID

### 2.4 Level 4: 端蓋 + 密封 + 外部特徵

#### 2.4.1 端蓋設計

| 端蓋 | 特徵 | 材料 |
|:-----|:-----|:-----|
| Drive side | 輸出軸通道、密封唇、車架鎖付座 | AZ91D + SUS304 嵌件 |
| Non-drive side | PCB 安裝 standoff、連接器開口、感測器線穿孔 | AZ91D |

#### 2.4.2 密封設計

| 密封位置 | 密封方式 | 規格 |
|:---------|:---------|:-----|
| 上下蓋分割面 | O-ring (NBR 或 FKM) | 依溝槽設計 |
| 輸出軸 | 油封 (TC type) | Ø_shaft × Ø_housing |
| 輸入軸 (踏板側) | 油封 or 防塵蓋 | Ø_shaft × Ø_housing |
| PCM 腔體 | 焊接密封 | 參照 WI-03 |
| 連接器 | IP67 防水連接器 | 依選型 |

#### 2.4.3 散熱鰭片

1. 殼體外表面設計淺鰭片：
   - 鰭片高度：3-5mm（不超出 Ø111mm 包絡）
   - 鰭片間距：5-8mm
   - 鰭片方向：軸向（配合騎行風向）
   - 鰭片僅在齒輪箱段外壁（壁厚充裕區域）
2. 不在馬達段設置鰭片（壁厚受限）

#### 2.4.4 車架鎖付座

1. e-Bike 中置馬達標準安裝：BB shell 鎖付
2. 鎖付座螺栓規格：M8 × 4（或依車架標準）
3. 鎖付座材料：SUS304 嵌件（承受反扭矩 125Nm）
4. 鎖付座位置：殼體下方，對稱分佈

## Step 3: 介面控制文件 (ICD) 定義

為每個關鍵配合面建立 ICD，確保跨 WI 的幾何一致性。

### 3.1 ICD-01: 馬達 ↔ 殼體

| 項目 | 規格 |
|:-----|:-----|
| 定位方式 | 定子外圈止口配合 (H7/h6) |
| 軸向定位 | 定子背面靠殼體內壁台階面 |
| 固定方式 | 壓入 + 結構膠 (Loctite 648 或等效) |
| 間隙 | 轉子外圈與殼體 ID 間隙 ≥ 0.5mm（避免干涉） |
| 電偶隔離 | 定子(SMC)與殼體(AZ91D)間鋁鐵電位差小，可直接接觸 |

### 3.2 ICD-02: 齒輪箱 ↔ 殼體

| 項目 | 規格 |
|:-----|:-----|
| 定位方式 | 圓形齒輪外圈止口 (H7/h6) |
| 軸向定位 | 圓形齒輪法蘭面靠殼體台階 |
| 固定方式 | M4 螺栓 × 6 (等角分佈) |
| 扭矩傳遞 | 圓形齒輪不旋轉，固定於殼體 |

### 3.3 ICD-03: PCB ↔ 端蓋

| 項目 | 規格 |
|:-----|:-----|
| 安裝方式 | M3 銅柱 standoff × 4 (等角分佈) |
| 高度 | standoff 高度 = 端蓋到 PCB 底面 (依元件決定) |
| 散熱 | FET 背面 thermal pad → 端蓋 (導熱矽膠墊) |
| 連接器 | 端蓋開口 + IP67 防水連接器 pass-through |
| 信號線 | 感測器信號 → PCB via 端蓋穿孔（密封膠保護） |

### 3.4 ICD-04: 殼體上蓋 ↔ 下蓋

| 項目 | 規格 |
|:-----|:-----|
| 分割面 | 馬達/齒輪箱段分界（或依 DFM 選擇最佳分模線） |
| 定位方式 | 止口 + 定位銷 × 2 |
| 固定方式 | M5 螺栓 × 6 (等角分佈) |
| 密封 | O-ring 溝槽 (FKM, shore 70A) |
| 對準精度 | 同心度 ≤ 0.05mm（影響馬達氣隙） |

## Step 4: 公差鏈分析

### 4.1 關鍵公差鏈

#### 公差鏈 1: 馬達氣隙均勻度

```
路徑: 轉子外圈 → 轉子-軸配合 → 軸-軸承配合 → 軸承-殼體配合 → 殼體-定子配合 → 定子內圈

目標: 氣隙變動量 ≤ ±0.1mm (名義 0.8mm → 0.7-0.9mm)

堆疊:
  轉子 OD 公差      : ±0.02mm (磁石+背鐵研磨)
  轉子-軸同心度     : ≤ 0.02mm (壓配)
  軸承徑向遊隙       : ≤ 0.01mm (P5 級)
  殼體軸承座同心度   : ≤ 0.03mm (壓鑄+機加工)
  殼體-定子配合      : ≤ 0.02mm (H7/h6)
  ─────────────────────
  RSS 合計          : √(0.02²+0.02²+0.01²+0.03²+0.02²) = 0.047mm
  
  單側最大偏移: 0.047mm << 0.1mm → Pass ✓
```

#### 公差鏈 2: 軸向堆疊

```
路徑: Drive-side 端蓋面(Datum B) → 齒輪箱 → 馬達 → PCB → 感測器 → Non-drive 端蓋面

目標: 總軸向 ≤ 92mm

堆疊:
  端蓋 A 厚度       : 10mm ± 0.2
  齒輪箱             : 28mm ± 0.1
  馬達               : 18mm ± 0.1
  PCB + standoff     : 8mm  ± 0.3
  感測器             : 5mm  ± 0.1
  端蓋 B 厚度       : 10mm ± 0.2
  ─────────────────────
  名義合計: 79mm
  最大: 79 + 0.2+0.1+0.1+0.3+0.1+0.2 = 80.0mm << 92mm
  
  裕量: 92 - 80 = 12mm → 充裕 ✓
```

## 附錄 A: 組裝順序建議

```
1. 殼體下蓋：安裝 SUS304 嵌件（壓配或鑄入）
2. 殼體下蓋：安裝輸出側軸承
3. 安裝齒輪箱子組件（圓形齒輪配合止口 → 螺栓固定）
4. 安裝馬達定子（止口配合 → 結構膠固定）
5. 安裝下轉子盤（軸上壓配 or 螺栓）
6. 安裝上轉子盤（注意磁石極性!）
7. 安裝推力軸承
8. 注入 PCM 石蠟（液態注入、等凝固、密封焊接）
9. 安裝感測器到踏板軸
10. 安裝 PCB 到上蓋 standoff
11. 合蓋：上蓋配合止口+O-ring → 螺栓鎖固
12. 安裝油封
13. 功能測試：手轉確認無干涉、感測器信號確認
```
