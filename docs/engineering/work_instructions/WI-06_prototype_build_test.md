# WI-06: 原型製作與測試程序

> **版本**: 1.0 | **日期**: 2026-04-21
> **負責域**: 系統整合
> **前置條件**: WI-01~05 設計完成, WI-07 物料到齊
> **產出物**: 測試計畫、治具規格、V1-V14 實測數據、測試報告
> **預估工時**: 文件 3 天 / 執行 8-12 週
> **TR Gate**: TR5 (Alpha 原型) → TR6 (Alpha 驗證)

---

## 溯源

本 WI 執行 TRIZ Session v2 的 14 項驗證計畫 (V1-V14)，對應所有 5 個 TC 的 Px 分離驗證。

---

## 原型策略: 四階段漸進式

```
Phase A: 馬達單獨 ─── V1, V2, V3, V4
         │ (最高風險先驗證)
         ▼
Phase B: 齒輪單獨 ─── V5, V6, V7, V8
         │
         ▼
Phase C: 熱系統單獨 ─ V10, V13
         │
         ▼
Phase D: 系統整合 ─── V9, V11, V12, V14
```

**理由**: 馬達(AFM)是 TR0 最低成熟度子系統 + Top 風險 R-001 所在；先驗證馬達可避免後續白做。

---

## Phase A: 馬達單獨測試

### 環境需求

| 設備 | 規格 |
|:-----|:-----|
| 扭矩測試台 (dynamometer) | 額定 ≥ 10Nm, 轉速 ≥ 3000rpm |
| 扭力感測器 | 精度 ±0.5%, 量程 10Nm |
| 直流電源 | 48V / 30A |
| 馬達控制器 | 可調 FOC 參數（或用 WI-05 PCB 原型） |
| 溫度量測 | K-type 熱電偶 ×4 + 熱像儀 |
| 力感測器 (軸向) | 測量推力軸承載荷 |

### V1: AFM 靜態扭矩

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 馬達鎖轉，施加額定電流，量測靜態扭矩 |
| 判定 | ≥ 5.0 Nm |
| 額外 | 掃描電流角 0°-30° 找最大扭矩角（MTPA 驗證） |

### V2: 氣隙磁通密度

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 空載旋轉 → 量測反 EMF → 反推 Bg |
| 計算 | Bg = Ke_measured / (Nph × Apole × p × ω_factor) |
| 判定 | Bg ≥ 0.8T (間接) |
| 額外 | 與 FEA 預測對比，偏差 < 10% |

### V3: 軸向力平衡（風險 R-001）

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 馬達安裝在推力感測器支架上，旋轉 + 施加電流 |
| 量測 | 軸向力 vs 電流、軸向力 vs 轉速 |
| 判定 | F_axial < 推力軸承額定 / 2 |
| 失敗對策 | 調整磁石位置 / 加大推力軸承 / 回 WI-01 重設計 |

### V4: SMC 鐵損

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 空載旋轉 @3000rpm，量測輸入功率 → 減去機械損 = 鐵損 |
| 備選 | 量熱法：密封馬達，量測溫升速率 → P = mCpΔT/Δt |
| 判定 | ≤ 20W @3000rpm |

### Phase A 判定

| 結果 | 動作 |
|:-----|:-----|
| V1-V4 全 Pass | → 進入 Phase B |
| V3 Fail (軸向力超標) | → **停止**，回 WI-01 重新設計 |
| V1 Fail (扭矩不足) | → 檢查繞線/磁石/氣隙，調整後重測 |
| V4 Fail (鐵損超標) | → 評估 Somaloy 等級替換 or 降極數 |

## Phase B: 齒輪單獨測試

### 環境需求

| 設備 | 規格 |
|:-----|:-----|
| 背對背齒輪測試台 | 額定 ≥ 150Nm |
| 噪聲量測 | 1/2" 麥克風 + 頻譜分析儀 |
| 無響室或半無響室 | ISO 3744 |
| 應變計 | 黏貼於柔輪齒根 |

### V5: 噪聲 (SPL)

| 項目 | 內容 |
|:-----|:-----|
| 方法 | ISO 3744，SPL @1m，齒輪箱在背對背台上運轉 |
| 工況 | 100Nm @100rpm (額定), 125Nm @短時 |
| 判定 | ≤ TQ HPR50 基準值（需先取得 HPR50 實測基線） |
| 量測 | 1/3 octave 頻譜 + overall dB(A) |

### V6: 齒輪效率

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 背對背測試台，量測輸入/輸出扭矩和轉速 |
| 工況 | 掃描: 50Nm-125Nm × 50-100rpm |
| 計算 | η = (T_out × ω_out) / (T_in × ω_in) |
| 判定 | ≥ 85% @額定工況 |

### V7: 柔輪疲勞壽命（風險 R-002）

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 加速壽命試驗：125Nm @3sec on / 7sec off，循環 |
| 目標 | ≥ 10,000 cycles 無裂紋 |
| 監測 | 應變計 real-time + 定期目視/磁粉探傷 |
| 失敗判定 | 任何裂紋 = Fail → 回 WI-02 |

### V8: 齒根應力

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 應變計黏貼於 CF-PEEK 剛輪齒根，量測 @125Nm |
| 判定 | ≤ 150 MPa |
| 與 FEA 對比 | 偏差 < 20% |

## Phase C: 熱系統測試

### 環境需求

| 設備 | 規格 |
|:-----|:-----|
| 殼體 + PCM（不含馬達/齒輪，用電加熱器模擬） | — |
| 加熱器 | 矽膠加熱片，90-120W |
| 熱電偶 | K-type ×8（殼體外壁、內壁、PCM 中心、環境） |
| 熱像儀 | FLIR or equiv. |
| DSC (差示掃描量熱) | PCM 循環穩定性 |

### V10: 溫升測試

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 電加熱器模擬 90W 連續 8min（Case 1）/ 120W 8min（Case 2）|
| 環境 | 靜止空氣 25°C（Case 1）/ 40°C（Case 2）|
| 量測 | 殼體各點溫度 vs 時間曲線 |
| 判定 | 殼體最高溫 ≤ 80°C (Case 1) |
| 額外 | 量測恢復時間（斷電後降至 57°C 和 40°C 的時間） |

### V13: PCM 循環穩定性

| 項目 | 內容 |
|:-----|:-----|
| 方法 | DSC 量測：初始 ΔH → 1000 次加熱/冷卻循環 → 再測 ΔH |
| 循環條件 | 30°C ↔ 70°C, 升降溫速率 5°C/min |
| 判定 | ΔH 衰退 ≤ 5% |
| 額外 | 目視檢查洩漏、稱重確認無損耗 |

### PCM 洩漏測試（風險 R-003）

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 組裝後充氣 0.2MPa，保持 10min |
| 判定 | 壓降 ≤ 0.01MPa |
| 時機 | Phase C 開始前必須通過 |

---

## Phase D: 系統整合測試

### 環境需求

| 設備 | 規格 |
|:-----|:-----|
| 完整組裝 Drive Unit | 含馬達+齒輪+殼體+PCM+PCB+感測器 |
| e-Bike 測試車架 | 模擬實際安裝條件 |
| CMM (三次元) | 尺寸量測 |
| 鹽霧箱 | ASTM B117 |
| EMI 測試設備 | 頻譜分析儀 + ADC 量測 |

### V9: 總重量

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 電子秤量測完整組裝件 |
| 判定 | ≤ 2500g (目標 ≤ 2350g) |
| 分解 | 逐件稱重 vs BOM 估算，記錄偏差 |

### V11: 尺寸

| 項目 | 內容 |
|:-----|:-----|
| 方法 | CMM 量測 |
| 判定 | OD ≤ 111mm, 軸向 ≤ 92mm, Spindle ≤ 153mm |

### V12: 腐蝕（長期測試）

| 項目 | 內容 |
|:-----|:-----|
| 方法 | ASTM B117 鹽霧 500hr |
| 判定 | 腐蝕深度 ≤ 0.05mm |
| 樣品 | 完整組裝件 × 3 |
| 時間 | 21 天（可與其他 Phase D 測試並行） |

### V14: EMI / ADC SNR

| 項目 | 內容 |
|:-----|:-----|
| 方法 | 馬達全速運轉 + 滿載，量測扭矩感測器 ADC 輸出 SNR |
| 判定 | SNR ≥ 60dB |
| 失敗對策 | 加厚 μ-metal、改善差動走線、調整 ADC 取樣時機 |
