## Step 1: 齒輪參數選定

### 1.1 齒比選定

1. 選擇減速比 i：推薦 **i = 26**
   - 理由：整數比、相對接近 1:25 目標、齒數相容性佳
2. 圓形齒輪齒數 Zcs = i × 2 + Zfs（典型 Zfs = Zcs - 2）
   - Zcs = 200, Zfs = 198 → i = (Zcs - Zfs) / Zfs × Zcs/2 ... 
   - 實際: 諧波齒輪比 = Zcs / (Zcs - Zfs)，若 Zcs=200, Zfs=198 → i = 200/2 = 100（太大）
   - **修正**: Zfs = Zcs - 2 → i = Zcs / 2
   - 要 i=26: Zcs = 52, Zfs = 50 → 檢查嚙合率和強度
   - 或 i=30: Zcs = 60, Zfs = 58

3. 推薦齒數組合：

| 減速比 | Zcs | Zfs | 模數 m | 齒輪 OD |
|:-------|:----|:----|:-------|:--------|
| 1:26 | 52 | 50 | 1.5 | Ø78mm |
| 1:30 | 60 | 58 | 1.2 | Ø72mm |
| 1:25 | 50 | 48 | 1.5 | Ø75mm |

4. 選定後記錄到齒輪參數表

### 1.2 齒形參數

| 參數 | 推薦值 | 備註 |
|:-----|:-------|:-----|
| 壓力角 | 20° (標準) | — |
| 齒頂修形 | 漸開線 + 齒頂圓弧修形 | 降低嚙合衝擊 |
| 全齒高 | 2.25m | 標準 |
| 齒頂間隙 | 0.25m | 標準 |

## Step 2: 柔輪設計與 FEA（風險 R-002）

> **安全關鍵項 — 柔輪疲勞是 Top 2 風險**

### 2.1 柔輪幾何

1. 杯形柔輪設計（最常見形式）：
   - 杯身長度：影響柔輪撓性和疲勞壽命
   - 壁厚：典型 0.3-0.5mm（齒圈段）
   - 隔膜厚度：杯底連接輸出法蘭

2. 關鍵尺寸依減速比和外徑約束計算

### 2.2 柔輪 FEA — 橢圓變形分析

1. 建立柔輪 3D 模型
2. 材料卡：[MC-03 CoCrMo F1537](../material_cards/MC-03_cocrmo_f1537.md)
   - E = 210-230 GPa
   - σ_y ≥ 1000 MPa (wrought, cold-worked)
   - ν = 0.29
3. 邊界條件：
   - 波發生器施加橢圓位移（長軸/短軸比 = 設計值）
   - 圓形齒輪約束齒圈外徑
   - 輸出法蘭施加扭矩載荷
4. 求解：靜態 + 循環載荷
5. 後處理：
   - 提取齒根最大 von Mises 應力 σ_max
   - 提取杯身最大交變應力 Δσ

### 2.3 疲勞分析 — Goodman Diagram

1. 取 CoCrMo F1537 S-N 曲線（查文獻或 MC-03）
2. 計算：
   - 平均應力 σ_m = (σ_max + σ_min) / 2
   - 交變應力 σ_a = (σ_max - σ_min) / 2
3. 在 Goodman diagram 上標點，確認安全係數：

```
σ_a / Se + σ_m / σ_UTS = 1/SF

目標: SF ≥ 3.0 (安全關鍵件)
```

4. 若 SF < 3.0：
   - 增加柔輪壁厚
   - 降低橢圓比（降低變形量，但影響嚙合率）
   - 表面拋光 Ra ≤ 0.4μm（提高疲勞壽命）
   - 考慮表面處理（噴丸強化）

### 2.4 判定

| 結果 | 動作 |
|:-----|:-----|
| SF ≥ 3.0 且齒根應力 < σ_y/2 | Pass |
| 2.0 ≤ SF < 3.0 | Marginal — 增加表面處理或降低峰值扭矩工況 |
| SF < 2.0 | **Fail** — 重新設計柔輪幾何或材料 |

## Step 3: 剛輪設計

### 3.1 剛輪 FEA — 齒根應力

1. 建立剛輪齒形 3D 模型
2. 材料卡：[MC-04 CF-PEEK 30%CF](../material_cards/MC-04_cfpeek_30cf.md)
   - E_flex = 22-28 GPa (依纖維方向)
   - σ_flex = 380 MPa @23°C
   - 注意異向性：沿射出流動方向強度較高
3. 施加載荷：30% 齒同時承載（諧波特性）
   - 單齒載荷 = 輸出扭矩 / (齒數 × 30% × 節圓半徑)
4. 判定：齒根 von Mises ≤ **150 MPa**（V8 判定標準，SF ≈ 2.5）

### 3.2 射出成型考量

| 項目 | 要求 |
|:-----|:-----|
| 模溫 | 180-200°C |
| 料溫 | 380-400°C |
| 澆口位置 | 非齒面、均勻填充 |
| 纖維取向 | 齒根區纖維需沿齒長方向（模流分析確認） |
| 收縮率 | CF-PEEK 約 0.1-0.3%（含纖維方向差異） |

---

## Step 4: 波發生器設計

### 4.1 軸承選型

1. 波發生器內裝薄截面軸承（承受橢圓預負荷）
2. 推薦類型：

| 類型 | 適用 | 建議 |
|:-----|:-----|:-----|
| 薄截面交叉滾子 (CSF/CSG) | 高剛性、精確 | **首選** |
| 四點接觸 | 承受徑向+軸向 | 備選 |
| 柔性軸承 (諧波專用) | 匹配橢圓變形 | 若可取得 |

2. 軸承規格需匹配：
   - ID = 踏板軸 OD + 配合間隙
   - OD = 柔輪 ID - 配合間隙
   - 寬度 ≤ 齒輪軸向預算

### 4.2 橢圓凸輪設計

1. 橢圓長/短軸比由嚙合量決定
2. 凸輪材料：SUJ2 (高碳鉻軸承鋼) 淬火 HRC 58-62
3. 凸輪表面精度：Ra ≤ 0.2μm

## Step 5: 效率估算

### 5.1 損耗分項

| 損耗來源 | 估算方法 | 典型占比 |
|:---------|:---------|:---------|
| 齒面滑動摩擦 | μ × F_n × v_slide | 50-60% |
| 柔輪遲滯 | 材料阻尼 × 變形能 | 15-20% |
| 波發生器軸承 | 軸承摩擦公式 | 15-20% |
| 潤滑攪拌 | 估算（少量油脂） | 5-10% |

### 5.2 效率計算

```
η_total = 1 - (P_friction + P_hysteresis + P_bearing + P_churning) / P_input

目標: η_total ≥ 85%
```

### 5.3 產出

效率值 → WI-03 熱模擬（齒輪箱損耗 = P_input × (1 - η)）

## Step 6: NVH 預測

### 6.1 嚙合頻率

```
f_mesh = Zfs × n_input / 60

例: Zfs=58, n_input=3000rpm → f_mesh = 2900 Hz
```

### 6.2 噪聲控制策略

| 策略 | 機制 | 本設計 |
|:-----|:-----|:-------|
| CF-PEEK 剛輪阻尼 | 高分子阻尼吸收嚙合振動 | ✓ 已設計 |
| 30% 齒同嚙 | 分散衝擊能量 | ✓ 諧波固有 |
| 油脂阻尼 | 齒面油脂薄膜吸振 | ✓ 需選擇適當油脂 |
| 殼體隔音 | AZ91D 殼體阻隔 | ✓ 已有 |

### 6.3 驗證方法

V5: ISO 3744 自由場 SPL @1m ≤ HPR50 基準（需取得 HPR50 實測基線值）
