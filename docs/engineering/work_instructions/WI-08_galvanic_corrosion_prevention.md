## Step 1: 風險評估

### 1.1 電偶電位差

```
AZ91D: E_corr ≈ -1.60 V (SCE)
SUS304: E_corr ≈ -0.10 V (SCE)

ΔE = 1.50 V → 嚴重電偶腐蝕風險（ΔE > 0.25V 即有風險）
```

### 1.2 腐蝕機制

- 陽極（被腐蝕）: AZ91D（鎂合金，更活潑金屬）
- 陰極（被保護）: SUS304（不鏽鋼）
- 在濕環境中，AZ91D 在 SUS304 接觸區域附近加速腐蝕
- 腐蝕速率與接觸面積比 (Ac/Aa) 成正比

### 1.3 使用環境

| 條件 | 等級 |
|:-----|:-----|
| 雨水/路面噴濺 | 經常 |
| 道路鹽分 | 季節性（冬季） |
| 汗水/清洗劑 | 偶爾 |
| 工作溫度 | 25-80°C（加速腐蝕動力學） |

## Step 2: 防護策略

### 2.1 三層防護體系

```
第一層: AZ91D 表面處理 — PEO 陶瓷層 ≥ 50μm
第二層: SUS304 嵌件絕緣隔離 — PEEK 墊圈/環氧膠
第三層: 系統密封 — IP67 整機防水
```

### 2.2 第一層: AZ91D 表面處理

| 工藝 | 膜厚 | 硬度 | 耐蝕性 | 推薦 |
|:-----|:-----|:-----|:-------|:-----|
| **PEO (Keronite)** | 50-100μm | 400-600 HV | 優（陶瓷層） | **首選** |
| DOW17 陽極氧化 | 25-50μm | 200-350 HV | 良 | 備選 |
| 化成處理 (HAE) | 5-25μm | — | 中 | 不推薦（太薄） |
| 電泳塗裝 | 20-30μm | — | 良（但不耐磨） | 外觀面補充 |

**PEO 規格要求**:
- 最小膜厚: 50μm（軸承座/嵌件區: 推薦 80μm）
- 硬度: ≥ 400 HV
- 鹽霧試驗: ASTM B117 ≥ 500hr 無紅鏽/白鏽

### 2.3 第二層: 嵌件絕緣隔離

#### 軸承座嵌件

```
截面示意:

  │ AZ91D 殼體 │
  │  PEO 層    │
  │  PEEK 套   │← 0.5-1mm 厚 PEEK 隔離套
  │  SUS304    │← 嵌件
  │  軸承外圈   │
```

1. PEEK 隔離套：管狀，套在 SUS304 嵌件外圍
2. PEEK 墊圈：端面隔離
3. 確保 SUS304 與 AZ91D 之間**無任何金屬直接接觸**

#### 鎖付座嵌件

```
螺栓 → PEEK 墊圈 → SUS304 嵌件 → 環氧結構膠 → PEO 層 → AZ91D
```

1. 螺栓本身也需絕緣：PEEK 墊圈 + PEEK 套管
2. 環氧結構膠 (Loctite EA 9461 或等效) 雙重功能：
   - 結構接著（剪切強度 ≥ 25 MPa）
   - 密封（阻隔水分進入接合面）

#### 壓鑄嵌入式

1. SUS304 嵌件表面預處理：
   - 噴砂 → PEO/陽極氧化處理（在嵌件表面形成氧化鋁陶瓷層）
   - 或：噴砂 → 環氧底漆 → 壓鑄包覆
2. 壓鑄後陶瓷層完整性檢查

### 2.4 第三層: 系統密封

| 密封位置 | 方式 | IP 等級 |
|:---------|:-----|:--------|
| 殼體上下蓋 | O-ring (FKM) | IP67 |
| 輸出軸 | TC 油封 | IP67 |
| 輸入軸 | TC 油封 / 防塵蓋 | IP67 |
| 連接器 | IP67 防水連接器 | IP67 |

## Step 3: 驗證程序

### 3.1 鹽霧試驗 (V12)

| 項目 | 規格 |
|:-----|:-----|
| 標準 | ASTM B117 |
| 時間 | 500 小時 |
| 溫度 | 35°C |
| NaCl 濃度 | 5% |
| 判定標準 | 腐蝕深度 ≤ 0.05mm (目視+截面量測) |
| 取樣 | 完整組裝件（含嵌件、密封） × 3 pcs |

### 3.2 加速腐蝕試驗（可選，更嚴格）

| 項目 | 規格 |
|:-----|:-----|
| 標準 | ASTM G85 (Cyclic Salt Spray) |
| 循環 | 鹽霧 → 乾燥 → 濕潤，168hr/cycle × 3 cycles |
| 判定 | 同 V12 |

### 3.3 PEO 層品質檢查

| 項目 | 方法 | 判定 |
|:-----|:-----|:-----|
| 膜厚 | 渦流測厚 | ≥ 50μm (嵌件區 ≥ 80μm) |
| 硬度 | 維氏微硬度 | ≥ 400 HV |
| 附著力 | 劃格試驗 (ISO 2409) | ≥ 4B |
| 孔隙率 | 電化學阻抗譜 (EIS) | Rp > 10⁵ Ω·cm² |
