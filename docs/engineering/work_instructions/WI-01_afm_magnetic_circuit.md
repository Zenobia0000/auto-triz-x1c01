## Step 1: 工作點定義

### 1.1 建立扭矩-轉速工作區間

繪製以下工作區間圖，標出額定點和峰值點：

```
扭矩 (Nm)
  7.0 ──── ┌──────┐ 峰值 (3sec burst)
            │      │
  5.0 ──── ├──────┼──────────┐ 額定 (continuous)
            │      │          │
            │      │          │
  0   ──── └──────┴──────────┴────── 轉速 (rpm)
            0    1000   2000   3000
                        ↑ base speed
```

### 1.2 確定設計點

| 工作點 | 扭矩 | 轉速 | 持續時間 | 約束 |
|:-------|:-----|:-----|:---------|:-----|
| **OP1 額定** | 5.0 Nm | 0-2000 rpm | 連續 8min | 溫升 ≤ 80°C |
| **OP2 峰值** | 7.0 Nm | 0-1500 rpm | 3 sec burst | 退磁安全 |
| **OP3 場弱** | 2.5 Nm | 2000-3000 rpm | 連續 | 不超過 DC bus 電壓 |

### 1.3 確定電氣約束

1. 查明 drive board 的 DC bus 電壓（典型 e-Bike: 36V 或 48V）
2. 確定最大相電流（連續 / 峰值），受限於 FET 額定和散熱
3. 記錄到設計目標表

## Step 2: 磁路初步計算

### 2.1 磁石選擇與分段

1. 開啟 [MC-01 NdFeB N42SH Material Card](../material_cards/MC-01_ndfeb_n42sh.md)
2. 記錄關鍵參數：
   - Br = 1.29-1.32 T (取 1.30 T 設計)
   - Hcj ≥ 1592 kA/m (SH grade, Tmax = 150°C)
   - BHmax ≈ 42 MGOe
3. **選擇磁石分段方式**：

| 方式 | 優點 | 缺點 | 推薦 |
|:-----|:-----|:-----|:-----|
| 周向分段 (circumferential segments) | 降低磁石渦流損、易製造 | 需更多磁石塊 | **推薦** |
| 整塊扇形 (solid sectors) | 磁石數少 | 高頻渦流損大 | 不推薦 |
| Halbach 排列 | 更高 Bg、減少背鐵 | 製造困難、成本高 | 備選 |
| 麵包塊 (bread-loaf) | 降低渦流 | 組裝困難 | 不推薦 |

4. 推薦：每極 2-3 段周向分段，磁石間隙 0.5mm（結構膠填充）

### 2.2 氣隙磁通密度計算

使用簡化磁路方程計算初始值：

```
Bg = Br × (lm / (lm + g × kc × (1 + σ_leak)))

其中:
  Br    = 1.30 T (磁石殘磁)
  lm    = 磁石厚度 (設計變量, 初始 3mm)
  g     = 機械氣隙 (初始 0.8mm, 含護套厚度)
  kc    = Carter 係數 (≈1.1 for open slots)
  σ_leak = 漏磁係數 (≈0.15 for AFM)

目標: Bg ≥ 0.8 T
```

1. 計算不同 lm (2.5mm, 3.0mm, 3.5mm) 下的 Bg
2. 驗證 Bg ≥ 0.8T 的最小 lm
3. 計算磁石體積和重量（密度 7.5 g/cm³）

### 2.3 定子齒/軛尺寸設計

1. 開啟 [MC-02 Somaloy 700 5P Material Card](../material_cards/MC-02_smc_somaloy700_5p.md)
2. 確定 SMC 最大工作磁通密度：Bsat ≈ 1.5T（建議設計值 Bmax ≤ 1.3T 保留裕量）
3. 計算齒寬：

```
齒寬 bt ≥ (Bg × τp × κ) / (Bmax_tooth × kstack)

其中:
  τp     = 極距 = π(Ro + Ri) / (2 × 2p)  (p=4, 取平均半徑)
  κ      = 磁通集中係數
  Bmax_tooth = 1.3 T (SMC 設計上限)
  kstack = 1.0 (SMC 無疊壓因素)
```

4. 計算軛厚：

```
軛厚 hy ≥ (Φ_pole / 2) / (Bmax_yoke × Lstk)

其中:
  Φ_pole = Bg × Apole (每極磁通)
  Bmax_yoke = 1.2 T
  Lstk   = 定子軸向厚度 (設計變量)
```

5. 繪製定子截面圖（俯視展開圖 + 軸向剖面圖）

## Step 3: 繞線設計

### 3.1 槽型與繞線方式

| 參數 | 選擇 | 理由 |
|:-----|:-----|:-----|
| 繞線方式 | 集中繞組 (concentrated) | 短端部、高槽滿率、適合 SMC 3D 磁路 |
| 層數 | 單層 (single-layer) | 簡化製造、降低相間絕緣風險 |
| 繞組配置 | 8P12S → q = 0.5 | 分數槽集中繞組、低齒槽扭矩 |

### 3.2 線徑與匝數計算

1. 確定反 EMF 常數 Ke 目標：

```
Ke = Vdc / (√3 × ωbase × km)

其中:
  Vdc    = DC bus 電壓 (48V)
  ωbase  = 基速轉速 (2000 rpm = 209.4 rad/s 電角速度 = 209.4×4=837.8 rad/s)
  km     = 調變指數裕量 (≈0.95)
```

2. 由 Ke 和 Bg 反推每相匝數 Nph
3. 由 Nph 和 12 槽分配，計算每槽匝數 Nc = Nph / (slots_per_phase / layers)
4. 確定線徑：

```
目標槽滿率 kfill = 0.45

線徑 d_wire = √(4 × Aslot × kfill / (π × Nc))

其中:
  Aslot = 槽面積 (從 Step 2.3 齒寬和槽深計算)
```

5. 從標準線規表選擇最近線徑（考慮 AWG 或 JIS 規格）
6. 計算相電阻 Rph 和銅損 Pcu = 3 × I² × Rph
7. 計算繞線端部突出量 — **確認不超過 18mm 軸向預算**

### 3.3 繞線規格表產出

## Step 4: 3D 電磁 FEA

### 4.1 軟體選擇

| 軟體 | 適用性 | 建議 |
|:-----|:-------|:-----|
| JMAG | AFM 專用求解器、SMC 材料庫完整 | **首選** |
| ANSYS Maxwell 3D | 通用、公司既有授權 | 備選 |
| COMSOL Multiphysics | 多物理場耦合（電磁+熱） | 如需熱耦合 |

### 4.2 幾何建模

1. 建立 1/4 模型（利用 8 極對稱性: 360°/8 = 45°，但 12 槽需 LCM(8,12)=24，取 1/4 = 90° 即 2 極 3 槽）
2. 建模順序：
   - 底部轉子盤（back iron + 磁石）
   - 氣隙（0.8mm）
   - 定子（SMC 齒+軛，12 槽中取 3 槽）
   - 氣隙（0.8mm）
   - 頂部轉子盤（back iron + 磁石，鏡像對稱）
3. 磁石方向：頂部和底部轉子的磁石極性需相對（N-N 或 S-S 面對面，形成軸向磁通穿過定子）

### 4.3 材料卡指定

| 區域 | 材料 | 材料卡來源 |
|:-----|:-----|:-----------|
| 轉子 back iron | S45C 或 DT4E 純鐵 | 軟體內建 |
| 磁石 | NdFeB N42SH | MC-01（匯入 B-H 第二象限曲線 + 溫度係數） |
| 定子齒+軛 | Somaloy 700 5P | MC-02（匯入 B-H 曲線 + 損耗曲線 W/kg vs f vs B） |
| 繞組 | 銅 (等效匝數設定) | 軟體內建，設定匝數和並繞數 |
| 氣隙 | Air | 軟體內建 |

### 4.4 網格設定

| 區域 | 網格尺寸 | 說明 |
|:-----|:---------|:-----|
| 氣隙 | **0.2mm** | 最關鍵區域，至少 4 層 |
| 磁石 | 0.5mm | 渦流損計算需足夠細 |
| 定子齒 | 0.5mm | 齒尖加密 |
| 定子軛 | 1.0mm | 磁通均勻區 |
| 轉子 back iron | 1.0mm | — |

### 4.5 邊界條件

| 邊界 | 設定 |
|:-----|:-----|
| 周向 | 週期性邊界 (anti-periodic for 1/4 model with 2 poles) |
| 軸向上下 | 自然邊界 (∂A/∂n = 0) |
| 外周 | 自然邊界或 balloon boundary |

### 4.6 求解設定

1. **靜態求解 (static)**：
   - 轉子位置掃描 0°~45°（電角度 0°~360°），步長 1° 電角度
   - 電流固定為額定 / 峰值
   - 產出：扭矩 vs 位置曲線 → 計算平均扭矩、扭矩漣波

2. **暫態求解 (transient)**：
   - 旋轉速度設定：1000rpm, 2000rpm, 3000rpm
   - 電流源驅動（MTPA 角度設定）
   - 求解時間 = 2 個電週期以上
   - 時間步長 ≤ 1/(20 × f_elec)
   - 產出：反 EMF 波形、扭矩波形、鐵損

### 4.7 參數掃描（設計優化）

建立以下參數掃描表：

| 掃描參數 | 範圍 | 步長 | 目的 |
|:---------|:-----|:-----|:-----|
| 磁石厚度 lm | 2.5-4.0mm | 0.5mm | 找最小 lm 使 Bg ≥ 0.8T |
| 機械氣隙 g | 0.6-1.0mm | 0.1mm | 平衡扭矩和製造可行性 |
| MTPA 電流角 γ | 0°-30° | 5° | 最大化磁阻扭矩利用 |
| 定子軸向厚度 | 6-10mm | 1mm | 平衡磁通和軸向空間 |

## Step 5: 後處理與判定

### 5.1 性能指標檢查

| 指標 | 目標 | 判定 |
|:-----|:-----|:-----|
| 平均扭矩 @OP1 | ≥ 5.0 Nm | Pass / Fail |
| 平均扭矩 @OP2 | ≥ 7.0 Nm | Pass / Fail |
| 扭矩漣波 | ≤ 10% peak-to-peak | Pass / Fail |
| 齒槽扭矩 | ≤ 3% 額定 (0.15 Nm) | Pass / Fail |
| 反 EMF THD | ≤ 15% | Info |
| 效率 @OP1 | ≥ 90% | Pass / Fail |

### 5.2 退磁檢查

1. 設定磁石溫度 = 120°C（最惡劣工況）
2. 施加 OP2 峰值電流（d 軸去磁方向）
3. 檢查磁石內部最低 B 值
4. 判定：最低 B > 磁石 knee point (約 0.3T @120°C for N42SH)
5. 若 Fail → 增加磁石厚度或降低電流角

### 5.3 反 EMF 波形品質

1. 提取空載反 EMF 波形（FEA 暫態, I=0, 額定轉速）
2. FFT 分析：記錄基波幅值和 5/7/11/13 次諧波
3. 計算 THD
4. 若 THD > 15%，考慮調整磁石弧度或偏心

## Step 6: 損耗分析與 Loss Map 產出

### 6.1 損耗分項

| 損耗類型 | 計算方法 | FEA 設定 |
|:---------|:---------|:---------|
| 銅損 Pcu | I²R × 溫度修正 (Rph @80°C ≈ 1.22 × Rph @25°C) | 解析計算 |
| 鐵損 Pcore | Steinmetz: Pcore = kh×f×B² + ke×(fB)² | FEA 後處理 (Somaloy loss curve) |
| 磁石渦流損 Pmag | FEA 暫態求解中計算 | 需磁石分段建模 |
| 機械損 Pmech | 軸承摩擦（估算） | 公式: Pmech ≈ 0.5 × μ × F × d × ω |

### 6.2 Loss Map 格式

建立以下格式的 loss map CSV，交付 WI-03 作為熱模擬輸入：

```csv
speed_rpm, torque_Nm, Pcu_W, Pcore_W, Pmag_W, Pmech_W, Ptotal_W, efficiency_pct
500, 5.0, XX, XX, XX, XX, XX, XX
1000, 5.0, XX, XX, XX, XX, XX, XX
...
3000, 2.5, XX, XX, XX, XX, XX, XX
```

產出範圍：
- 轉速：500, 1000, 1500, 2000, 2500, 3000 rpm
- 扭矩：1.0, 2.5, 5.0, 7.0 Nm (7.0 只到 1500rpm)

### 6.3 損耗預算判定

| 損耗 | 預算 | 判定 |
|:-----|:-----|:-----|
| 鐵損 @3000rpm 空載 | ≤ 20 W | Pass / Fail (V4) |
| 總損耗 @OP1 (5Nm, 2000rpm) | ≤ 100 W (η ≥ 90%) | Pass / Fail |
| 磁石渦流損 @3000rpm | ≤ 5 W (周向分段後) | Info |

**若鐵損超標**：
1. 比較 Somaloy 700 5P vs 700HR vs 500 的損耗曲線
2. 考慮降低極數到 6 極（電頻率降 25%，但扭矩漣波可能增加）
3. 考慮加大定子截面降低 Bmax（但增加軸向厚度）

## Step 7: 軸向力平衡檢查（風險 R-001）

> **本步驟為安全關鍵項 — 必須通過才能進入 TR1**

### 7.1 理想情況

雙轉子結構中，上下轉子對定子的軸向磁拉力理論上對稱抵消：

```
F_axial_net = F_top - F_bottom ≈ 0  (理想對稱)
```

### 7.2 公差敏感度分析

1. 在 FEA 中設定以下偏移情境：

| 情境 | 上氣隙 | 下氣隙 | 模擬 |
|:-----|:-------|:-------|:-----|
| 名義 | 0.80mm | 0.80mm | baseline |
| 定子偏上 0.1mm | 0.70mm | 0.90mm | worst case 1 |
| 定子偏下 0.1mm | 0.90mm | 0.70mm | worst case 2 |
| 磁石厚度差 0.1mm | lm-0.05 | lm+0.05 | worst case 3 |

2. 記錄每個情境的 F_axial_net (N)

### 7.3 推力軸承選型

1. 取 worst case 的 F_axial_max
2. 安全係數 ≥ 2.0
3. 所需推力軸承動額定: C_axial ≥ F_axial_max × 2.0 × L10 壽命修正

推薦軸承類型：
- 薄截面推力滾珠軸承 (e.g., NSK 51100 系列)
- 或整合到角接觸軸承中（雙列角接觸同時承受徑向+軸向）

4. 記錄軸承型號、尺寸、額定 → 交付 WI-04

### 7.4 判定

| 結果 | 動作 |
|:-----|:-----|
| F_axial_max < 軸承額定 / 2 | Pass → 繼續 |
| F_axial_max ≈ 軸承額定 / 2 | Marginal → 提高組裝精度要求 |
| F_axial_max > 軸承額定 | **Fail** → 回到 Step 2 增加氣隙或重新設計磁路 |
