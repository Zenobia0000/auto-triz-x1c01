> **版本**: 1.0 | **日期**: 2026-04-21

> **FEA 軟體**: JMAG, ANSYS Maxwell, COMSOL AC/DC Module

---

## 材料概述

NdFeB 稀土永磁，等級 N42SH（High Temperature 等級）。SH 等級可承受最高工作溫度 150°C，適用於中置驅動馬達轉子。本設計採用 8 極雙轉子架構，每極圓周分段 2-3 片以抑制渦電流損耗。

- **標準/等級**: N42SH (IEC 60404-8-1 / 中國 GB/T 13560)
- **製造商參考**: Shin-Etsu N42SH / TDK NEOREC42SH 或同等品
- **最大工作溫度**: 150°C

供 JMAG/ANSYS Maxwell 輸入之典型 N42SH 退磁曲線數據點：

| H (kA/m) | B (T) |
|:---|:---|
| 0 | 1.30 |
| -200 | 1.04 |
| -400 | 0.77 |
| -600 | 0.51 |
| -800 | 0.25 |
| -955 | 0.00 |

> **注意**: 以上為 20°C 線性近似。高溫退磁曲線呈現「膝點」(knee point)，150°C 時膝點 B 值約 0.3T。JMAG 應載入廠商提供的多溫度退磁曲線。

## FEA 輸入指引

### JMAG

1. **Material 設定**: 選擇 `Permanent Magnet` → `NdFeB` 類別
2. **退磁曲線**: 載入廠商 `.jmag-mat` 或手動輸入上表 B-H 數據
3. **溫度相依**: 啟用 `Temperature Dependent` → 輸入 α(Br)=-0.12%/°C, β(Hcj)=-0.55%/°C
4. **渦電流**: 設定電阻率 ρ=1.5×10⁻⁶ Ω·m，分段建模（每極 2-3 段）
5. **磁化方向**: 使用 `Parallel` 或 `Radial` 取決於磁極拓撲，8 極設定
6. **Mesh**: 磁石至少 3 層元素厚度，周向分段間隙 0.1mm

### ANSYS Maxwell

1. **Material**: 新增 `User-defined` → `Permanent Magnet`
2. **Demagnetization curve**: 輸入多溫度 B-H 曲線（20°C, 80°C, 120°C, 150°C）
3. **Conductivity**: σ = 1/(1.5×10⁻⁶) ≈ 6.67×10⁵ S/m
4. **Core Loss**: 啟用 `Eddy Current` loss，確認分段幾何正確
5. **Motion Setup**: 轉子帶磁石整體旋轉

### COMSOL AC/DC

1. **Domain**: 選擇 `Magnetic Fields (mf)` → `Permanent Magnet` node
2. **Constitutive relation**: `Remanent flux density` model
3. **Br vector**: 設定為 1.30 T，方向依極性交替
4. **μ_rec**: 1.05 (relative permeability in recoil)
5. **Electrical conductivity**: 6.67×10⁵ S/m（渦電流計算）

## 注意事項

1. **不可逆退磁風險**: 150°C 工況下，若反向磁場超過膝點，將發生不可逆退磁。堵轉工況需特別檢查。FEA 中應驗證磁石工作點始終在膝點以上。
2. **溫度計算順序**: 先跑電磁-熱耦合穩態，取得磁石溫度分佈後，用該溫度更新退磁曲線重新計算。
3. **分段效果**: 圓周分段 2-3 片/極可降低渦電流損約 50-70%。FEA 中必須建模分段間隙（通常 0.1mm 環氧填充）。
4. **脆性**: 燒結 NdFeB 極脆，結構 FEA 中應確認離心力下拉伸應力 < 30 MPa（含安全因子）。
5. **腐蝕**: 裸露 NdFeB 極易氧化，實際產品需 NiCuNi 或環氧塗層，FEA 幾何可忽略塗層厚度（~10-25μm）。
6. **稀土成本**: N42SH 含重稀土 Dy/Tb ~3-5wt%，成本敏感。設計變更前評估是否可降級至 N38SH。
