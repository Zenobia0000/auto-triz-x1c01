## 材料概述

鎂鋁鋅合金 AZ91D，壓鑄 (die-cast) 等級。密度僅 1810 kg/m³，為最輕的結構金屬合金之一。用於驅動單元外殼 (shell)，分為上下兩半殼體。壓鑄成型，建議使用真空壓鑄 (vacuum die-casting) 以減少氣孔率。外殼與 SUS304 軸承座嵌件及安裝螺柱組合使用。

- **標準**: ASTM B94 / EN 1753 (MgAl9Zn1)
- **化學成分 (wt%)**: Al 8.5-9.5%, Zn 0.45-0.9%, Mn 0.17-0.4%, Si <0.05%, Cu <0.015%, Ni <0.001%, Fe <0.004%, Mg bal.
- **成型方式**: 高壓壓鑄 (HPDC)，建議真空輔助
- **鑄態**: as-cast (F temper)，無需後續熱處理

## FEA 輸入指引

### ANSYS Mechanical（殼體結構分析）

1. **Material Model**: `Isotropic Elasticity` → E=45 GPa, ν=0.35, ρ=1810 kg/m³
2. **塑性** (若需):
   - Bilinear Isotropic Hardening: σ_y=160 MPa, E_t=1500 MPa
3. **載荷工況**:
   - 靜態: 馬達反力矩 125 Nm 經安裝點傳遞、軸承座反力
   - 振動: 車輛振動載荷 (e-Bike EN 15194 相關)
   - 熱應力: 馬達發熱 → 殼體溫度梯度 → 與 SUS304 嵌件 CTE 差異應力
4. **接觸**:
   - 與 SUS304 嵌件: `Bonded` (壓鑄包鑄) 或 `Frictional` (壓入配合)
   - 上下殼螺栓接合: `Frictional` contact + `Bolt pretension`
5. **Mesh**: 最小壁厚 2.0mm 處至少 3 層 hex 元素
6. **壓鑄缺陷考量**: 在應力較高區域，降伏強度乘以 0.7-0.8 的缺陷因子

### COMSOL Structural Mechanics

1. **Material**: `User-defined` → 輸入等向性彈性常數
2. **Physics**: `Solid Mechanics` → `Linear Elastic Material`
3. **Thermal Stress**: 耦合 `Heat Transfer in Solids` → 匯入溫度場做熱應力分析

### 壓鑄製程 FEA (ProCAST / MAGMASOFT)

1. **模具材料**: H13 tool steel
2. **澆鑄溫度**: 650-680°C
3. **模具溫度**: 180-220°C (預熱)
4. **充填時間**: < 50 ms (高壓鑄造)
5. **真空度**: < 50 mbar (真空壓鑄)
6. **關鍵輸出**: 縮孔位置、氣孔分佈、最後凝固區 → 避免在軸承座/螺栓座等受力區域

## 注意事項

1. **電偶腐蝕 (Galvanic Corrosion)**: Mg 為最活潑的結構金屬，與 SUS304 不鏽鋼接觸時電位差 ΔE ≈ 1.5V，極易發生嚴重電偶腐蝕。**必須**使用 PEO (Plasma Electrolytic Oxidation, 如 Keronite) 表面處理 ≥50μm，並在接觸面使用絕緣墊片或塗層隔離。
2. **PEO 表面處理**: Keronite PEO 膜層厚度 50-80μm，硬度 ~400-600 HV，耐蝕性 >1000h 中性鹽霧試驗。FEA 幾何中可忽略膜層厚度，但熱分析中膜層熱阻 (k ≈ 1.5 W/mK) 可能需考慮。
3. **蠕變限制**: AZ91D 在 120°C 以上蠕變顯著。殼體中軸承座、螺栓預緊力區域若溫度 >100°C，需使用 SUS304 嵌件承受載荷，Mg 殼僅作為外殼包覆。
4. **熱膨脹配合**: CTE=26 μm/m°C 遠高於 SUS304 (CTE~17 μm/m°C) 和鋼製軸承 (CTE~12 μm/m°C)。高溫下 Mg 殼膨脹大於嵌件，配合可能鬆動。需計算全溫度範圍內的配合量。
5. **可燃性**: 鎂合金切屑/粉塵可燃。加工時使用乾切削或專用冷卻液，嚴禁水基冷卻液。
6. **回收性**: AZ91D 可 100% 回收再利用，有利於產品永續性評估。
7. **EMI 屏蔽**: 鎂合金殼體提供中等程度的電磁屏蔽效果，有助於 EMC 合規。
