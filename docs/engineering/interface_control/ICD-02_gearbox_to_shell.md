## 介面概述

諧波減速器 (OD ≤ 90mm, 28mm axial) 與 AZ91D 殼體之間的定位、固定與扭矩傳遞介面。圓形齒輪 (rigid spline, CF-PEEK) 固定於殼體作為反力構件，承受高達 125Nm 反力矩。柔輪 (flex spline, CoCrMo) 透過波產生器旋轉輸出至鏈輪。

```

              │                           │
    ──────────┼───────────────────────────┼──────────
              │  台階面 (軸向定位)        │
              │  ┌── Rigid Spline ─────┐  │
              │  │   CF-PEEK           │  │
              │  │   止口 H7/h6        │  │
              │  │   M4×6 螺栓固定     │  │
              │  │                     │  │
              │  │  ┌─ Flex Spline ─┐  │  │
              │  │  │  CoCrMo       │  │  │

              │  │  │ ┌Wave Gen.──┐ │  │  │
```

## 配合規格

| 項目 | 規格 | 備註 |
| :--- | :--- | :--- |
| 定位方式 | 圓形齒輪外圈止口 H7/h6 | Rigid spline OD → 殼體 ID |
| 軸向定位 | 圓形齒輪法蘭面靠殼體台階 | 台階面平面度 ≤ 0.02mm |
| 固定方式 | M4 螺栓 × 6 (等角分佈, 60° 間距) | 鍍鋅鋼螺栓, 防鬆螺紋膠 |
| 齒輪箱 OD | ≤ 90mm | — |
| 齒輪箱軸向長度 | 28mm | — |
| 反力矩 | ≤ 125Nm | 圓形齒輪承受，經螺栓傳至殼體 |
| 柔輪材料 | CoCrMo | 高疲勞壽命 |
| 圓形齒輪材料 | CF-PEEK | 碳纖維強化聚醚醚酮 |
| 波產生器軸承 | 交叉滾子軸承 (Crossed Roller) | — |
| 輸出方式 | 輸出法蘭 → 鏈輪 (花鍵連接) | — |
| 潤滑 | 封閉油脂 (Kluber 或等效) | 密封於齒輪箱腔體內 |
| 殼體材料 | AZ91D | PEO 表面處理 ≥ 50μm |

## 公差與 GD&T

| 公差項目 | 規格 | 方法 |
| :--- | :--- | :--- |
| 止口配合公差 | H7/h6 | 標準 ISO 配合 |
| 台階面平面度 | ≤ 0.02mm | CMM 量測 |
| 螺栓孔位置度 | ⌀0.1mm (PCD 基準) | CMM 量測 |
| 圓形齒輪法蘭面平行度 | ≤ 0.03mm to 止口軸線 | CMM 量測 |
| 輸出法蘭同心度 | ⌀0.05mm to 殼體軸承座 | 組裝後量測 |
| M4 螺栓鎖固扭矩 | 依螺栓規格 (預估 3.5~4.5 Nm) | 扭矩扳手 |

## 熱/電/密封 路徑

### 密封

- 齒輪箱腔體為獨立密封空間
- 油脂封閉於腔體內，防止外漏汙染馬達或 PCB
- 密封方式: 圓形齒輪法蘭面 O-ring + 輸出法蘭油封

### 潤滑

- 油脂: Kluber 或等效長壽命合成油脂
- 免維護設計，壽命期間無須補充

### 扭矩傳遞路徑

```
馬達軸 → 波產生器 → 柔輪齒嚙合 → 圓形齒輪(固定, 反力) → M4 螺栓 × 6 → 殼體
                                  → 輸出法蘭 → 花鍵 → 鏈輪
```

## 驗收條件

| 驗收項目 | 方法 | 判定基準 |
| :--- | :--- | :--- |
| 止口配合 | 塞規 / CMM | H7/h6 公差帶內 |
| 螺栓鎖固扭矩 | 扭矩扳手 | 規格值 ±10% |
| 反力矩測試 | 靜態扭矩測試 | 125Nm 無滑動或變形 |
| 油脂密封 | 組裝後靜置 48hr 觀察 | 無洩漏 |
| 輸出法蘭同心度 | 千分錶打表 | ⌀0.05mm 內 |
| 背隙測試 | 扭轉背隙量測 | 依諧波減速器規格 |
| 噪音測試 | 空載運轉 | ≤ 設計限值 (dBA) |

## 變更管控

- 螺栓規格或數量變更須重新驗證反力矩承載能力
- 油脂型號變更須確認與 CoCrMo / CF-PEEK / AZ91D 材料相容性
- 止口配合公差變更須同步更新輸出法蘭同心度分析
- 齒輪箱 OD 變更影響殼體 ID，須同步更新 ICD-01 馬達段間隙
