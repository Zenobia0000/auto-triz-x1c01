## 介面概述

AFM 雙轉子馬達 (Ø100mm, 18mm axial) 與 AZ91D 殼體 (OD111mm, 92mm axial) 之間的定位、固定與熱傳介面。定子外圈以止口配合壓入殼體內壁，轉子經氣隙與殼體隔離。此介面為馬達散熱主要路徑。

## 配合規格

| 項目 | 規格 | 備註 |
| :--- | :--- | :--- |
| 定位方式 | 定子外圈止口配合 H7/h6 | 定子 OD → 殼體 ID |
| 軸向定位 | 定子背面靠殼體內壁台階面 | 台階面平面度 ≤ 0.02mm |
| 固定方式 | 壓入 + 結構膠 (Loctite 648 或等效) | 高強度圓柱固持膠 |
| 馬達外徑 | Ø100mm | 定子與轉子共用 |
| 馬達軸向長度 | 18mm | 含雙轉子 + 定子 |
| 殼體 ID (馬達段) | ≈ 102mm | 配合止口處 |
| 殼體 OD | 111mm | 含 PCM 腔體 |
| 殼體壁厚 | ~4.5mm | 含 PEO 塗層 |
| 轉子-殼體間隙 | ≥ 0.5mm | 轉子外圈至殼體 ID |
| 氣隙 (每側) | 0.8mm | 轉子與定子間 |
| 定子材料 | SMC Somaloy 700 5P | 軟磁複合材料 |
| 殼體材料 | AZ91D (鎂合金) | PEO 表面處理 ≥ 50μm |
| 電偶隔離 | 不需要 | SMC 與 AZ91D 電位差小，可直接接觸 |

## 公差與 GD&T

| 公差項目 | 規格 | 方法 |
| :--- | :--- | :--- |
| 氣隙均勻度 | ±0.1mm | RSS 公差鏈分析結果 = 0.047mm |
| 殼體軸承座孔位置度 | ⌀0.03mm to Datum A | CMM 量測 |
| 殼體上下蓋止口同心度 | ⌀0.05mm | CMM 量測 |
| 止口配合公差 | H7/h6 | 標準 ISO 配合 |
| 台階面平面度 | ≤ 0.02mm | — |
| 定子壓入力 | 依膠合規格 | 製程管控 |

## 熱/電/密封 路徑

### 熱路徑

```
定子發熱 (SMC) → 止口接觸面 → 殼體內壁 (AZ91D) → PCM 腔體 (RT55) → 殼體外壁 → 自然對流
```

- 止口接觸面為主要熱傳介面，接觸面積最大化
- 結構膠 (Loctite 648) 同時填充微觀間隙提升熱導
- PCM (RT55, Tm ≈ 55°C) 提供暫態熱緩衝

### 電路徑

- 定子 (SMC) 與殼體 (AZ91D) 直接接觸
- 鋁鐵電位差小，無電偶腐蝕風險
- 殼體接地路徑經由車架連接

## 驗收條件

| 驗收項目 | 方法 | 判定基準 |
| :--- | :--- | :--- |
| 止口配合 | 塞規 / CMM | H7/h6 公差帶內 |
| 氣隙均勻度 | 旋轉氣隙感測器 | ±0.1mm 內 |
| 軸承座位置度 | CMM | ⌀0.03mm 內 |
| 上下蓋同心度 | CMM | ⌀0.05mm 內 |
| 壓入力紀錄 | 壓入治具力感測器 | 依製程規格 |
| 膠合固化 | 固化時間 + 拉拔測試 | 依 Loctite 648 TDS |
| 熱阻測試 | 溫升實驗 | 定子至殼體外壁 ΔT 在設計餘裕內 |

## 變更管控

- 任何止口配合公差變更須同步更新公差鏈分析 (RSS)
- 氣隙規格變更須經馬達電磁設計審查
- 結構膠型號變更須重新驗證拉拔強度與熱阻
- 殼體 ID 變更影響 PCM 腔體容積，須同步更新熱管理分析
