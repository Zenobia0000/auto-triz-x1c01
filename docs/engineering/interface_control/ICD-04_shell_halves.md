## 介面概述

AZ91D 殼體上蓋與下蓋之間的分割、定位、固定與密封介面。分割面位於馬達段與齒輪箱段交界處。兩半殼體合攏後須維持馬達氣隙均勻度，並密封 PCM 腔體。

## 配合規格

| 項目 | 規格 | 備註 |
| :--- | :--- | :--- |
| 分割面位置 | 馬達段 / 齒輪箱段分界 | — |
| 定位方式 | 止口 + 定位銷 × 2 | 定位銷直徑依設計 (預估 Ø4~Ø5mm) |
| 固定方式 | M5 螺栓 × 6 (等角分佈, 60° 間距) | 防鬆螺紋膠 |
| 密封方式 | O-ring 溝槽 | FKM (氟橡膠), Shore 70A |
| 對準精度 | 同心度 ≤ 0.05mm | 確保馬達氣隙均勻 |
| 上蓋材料 | AZ91D 壓鑄 | PEO 處理 ≥ 50μm |
| 下蓋材料 | AZ91D 壓鑄 | PEO 處理 ≥ 50μm |
| 殼體總軸向長度 | 92mm | 上蓋 + 下蓋合計 |
| 殼體 OD | 111mm | — |
| PCM 腔體 | 跨越上下蓋 | 注入/排氣口焊接密封 |

## 公差與 GD&T

| 公差項目 | 規格 | 方法 |
| :--- | :--- | :--- |
| 上下蓋同心度 | ⌀0.05mm | CMM 量測 (合攏狀態) |
| 止口配合公差 | H7/h6 或 H7/js6 | 標準 ISO 配合 |
| 分割面平面度 | ≤ 0.03mm | CMM 量測 |
| 定位銷孔位置度 | ⌀0.02mm to Datum A | CMM 量測 |
| O-ring 溝槽尺寸 | 依 O-ring 標準 (AS568 或 JIS) | 溝槽深度/寬度公差依規範 |
| M5 螺栓孔位置度 | ⌀0.15mm (PCD 基準) | CMM 量測 |
| PEO 塗層厚度 | ≥ 50μm | 渦電流測厚 |
| 組裝後氣隙均勻度 | ±0.1mm (與 ICD-01 一致) | 旋轉氣隙感測器 |

## 熱/電/密封 路徑

### 密封

| 密封位置 | 方式 | 備註 |
| :--- | :--- | :--- |
| 上下蓋分割面 | FKM O-ring, Shore 70A | 連續溝槽環繞整個分割面 |
| PCM 注入口 | 焊接密封 | PCM 注入後永久封閉 |
| PCM 排氣口 | 焊接密封 | PCM 注入後永久封閉 |

### 熱路徑

- PCM 腔體 (RT55) 跨越上下蓋，兩半殼體共同提供散熱面積
- 分割面 O-ring 不中斷 PCM 腔體連通性 (PCM 腔體在 O-ring 內側)
- 上下蓋壁面均為散熱路徑的一部分

### 電路徑

- 上下蓋經金屬止口直接接觸，確保電氣連續性
- 殼體接地路徑不因分割面中斷

## 驗收條件

| 驗收項目 | 方法 | 判定基準 |
| :--- | :--- | :--- |
| 上下蓋同心度 | CMM (合攏後) | ⌀0.05mm 內 |
| 分割面平面度 | CMM (單件) | ≤ 0.03mm |
| 定位銷配合 | 手動插入 + 塞規 | 順暢插入，無晃動 |
| O-ring 壓縮率 | 溝槽尺寸量測計算 | 15~25% 壓縮率 |
| 螺栓鎖固扭矩 | 扭矩扳手 | 依 M5 規格 (預估 5~7 Nm) |
| 防水測試 | IPX7 (1m, 30min) | 無滲水 |
| 氣隙均勻度 (組裝後) | 旋轉氣隙感測器 | ±0.1mm 內 |
| PCM 腔體密封 | 氦檢漏 / 壓力衰減 | 洩漏率 ≤ 規格值 |
| PEO 塗層 | 渦電流測厚 | ≥ 50μm |

## 變更管控

- 分割面位置變更為重大設計變更，影響所有子系統介面 (ICD-01, ICD-02, ICD-03)
- O-ring 材質或截面變更須重新驗證密封性能與壓縮率
- 定位銷規格變更須重新驗證同心度與氣隙均勻度
- 螺栓數量或規格變更須重新驗證結構剛性 (FEA)
- PCM 腔體幾何變更須同步更新熱管理模擬
- PEO 處理規格變更須評估耐蝕性與配合公差影響
