# ICD-03: PCB to End Cap Interface

> **版本**: 1.0 | **日期**: 2026-04-21
> **相關 WI**: WI-04, WI-05 (Electronics Design), WI-02 (Shell Design)
> **負責簽核**: 電子 / 機構 工程師

---

## 介面概述

環形 PCB (8mm axial) 與非驅動側端蓋 (AZ91D) 之間的安裝、散熱、防水與信號穿越介面。PCB 經銅柱 standoff 固定於端蓋，FET 散熱透過導熱矽膠墊傳導至端蓋，外部連接器經 IP67 防水穿越。

```
         外部連接器 (IP67)
              │
    ══════════╪══════════════════════════  ← 端蓋外表面
    ║  AZ91D  │  End Cap               ║
    ║         │  (Non-drive side)       ║
    ║   ┌─────┴─────┐                  ║
    ║   │ IP67 Conn. │  信號線穿孔      ║
    ║   │ Pass-thru  │  (密封膠)        ║
    ║   └─────┬─────┘                  ║
    ══════════╪══════════════════════════  ← 端蓋內表面
              │
         ┌────┴────┐
         │Thermal  │ ← 導熱矽膠墊
         │Pad      │   (FET 背面)
         └────┬────┘
    ┌─────────┼─────────────────────────┐
    │   ┌─────┴─────┐                  │
    │   │   FET     │   Annular PCB    │
    │   │           │   OD ≤ shell     │
    │   └───────────┘   ID - 2mm       │  8mm
    │                   ID ≥ 34mm      │  axial
    │   ○ M3 standoff (×4, 等角)       │
    └───────────────────────────────────┘
         │              │
    ┌────┴────┐    ┌────┴────┐
    │ Standoff│    │ Standoff│  ← M3 銅柱
    │ (×4)   │    │ (×4)   │
    └────┬────┘    └────┬────┘
    ═════╧═══════════════╧═══════════════  ← 端蓋安裝面
                  (截面圖)
```

## 配合規格

| 項目 | 規格 | 備註 |
| :--- | :--- | :--- |
| 安裝方式 | M3 銅柱 standoff × 4 | 等角分佈 (90° 間距) |
| Standoff 高度 | 端蓋內表面到 PCB 底面距離 | 依詳細設計確定 |
| PCB 形式 | 環形 (Annular) | — |
| PCB OD | ≤ 殼體 ID - 2mm | 留邊間隙 ≥ 1mm/side |
| PCB ID | ≥ 34mm | 踏板軸 OD (32mm) + 2mm 間隙 |
| PCB 軸向厚度 | 8mm (總高含元件) | — |
| FET 散熱 | Thermal pad → 端蓋 | 導熱矽膠墊 |
| 外部連接器 | IP67 防水連接器 | 端蓋開口 pass-through |
| 信號線穿越 | 端蓋穿孔 + 密封膠保護 | 感測器信號至 PCB |
| 端蓋材料 | AZ91D | PEO 表面處理 ≥ 50μm |

## 公差與 GD&T

| 公差項目 | 規格 | 方法 |
| :--- | :--- | :--- |
| Standoff 螺孔位置度 | ⌀0.15mm (PCD 基準) | CMM 量測 |
| PCB 安裝孔位置度 | ⌀0.15mm | PCB 製程管控 |
| Standoff 高度公差 | ±0.1mm | 確保導熱墊壓縮量一致 |
| 端蓋連接器開口位置 | ±0.3mm | 模具管控 |
| 端蓋平面度 (PCB 安裝區) | ≤ 0.05mm | CMM 量測 |

## 熱/電/密封 路徑

### 熱路徑

```
FET 發熱 → Thermal Pad (導熱矽膠墊) → 端蓋 (AZ91D) → 外部對流
```

- 導熱矽膠墊厚度依 standoff 高度與 FET 封裝高度決定
- 墊片壓縮率 10~20% 以確保接觸
- 端蓋外表面可設計散熱鰭片增加散熱面積

### 電路徑

- 感測器信號線經端蓋穿孔至 PCB
- 穿孔處以密封膠 (RTV 或等效) 保護，兼防水與絕緣
- 外部線束經 IP67 連接器進入

### 密封

| 密封位置 | 方式 | 等級 |
| :--- | :--- | :--- |
| 外部連接器 | IP67 防水連接器 | IP67 |
| 信號線穿孔 | 密封膠 (RTV 或等效) | IP67 |
| 端蓋與殼體接合 | O-ring (參見 ICD-04) | IP67 |

## 驗收條件

| 驗收項目 | 方法 | 判定基準 |
| :--- | :--- | :--- |
| Standoff 安裝高度 | 高度規量測 | 規格值 ±0.1mm |
| PCB 固定穩固度 | 手動檢查 + 振動測試 | 無鬆動 |
| 導熱墊接觸 | 壓痕觀察 / 熱像儀 | 完全接觸，無空隙 |
| FET 溫升 | 滿載溫升測試 | FET Tj ≤ 額定值 |
| IP67 防水 | IPX7 浸水測試 (1m, 30min) | 無滲水 |
| 連接器插拔 | 插拔壽命測試 | ≥ 1000 次 |
| 信號完整性 | 感測器信號波形量測 | 符合設計規格 |

## 變更管控

- Standoff 高度變更須同步更新導熱墊規格與壓縮率計算
- PCB OD/ID 變更須確認不干涉殼體內壁或踏板軸
- 連接器型號變更須重新驗證 IP67 等級
- 端蓋穿孔位置變更須更新密封膠塗佈 SOP
