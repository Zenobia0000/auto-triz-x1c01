# TRIZ 物理矛盾分離原則

> **用途**：Step 5a-2，當矛盾為物理矛盾（同一物件需同時具備矛盾屬性）時，使用分離原則找解法方向。
> **設計哲學**：每個策略以**第一性原理**演繹，標註控制方程/物理定律、量化實例、跨域案例。
> **注入方式**：全量注入 Prompt 上下文。

## 什麼是物理矛盾？

物理矛盾 = 同一個系統參數需要同時滿足兩個相反的要求。

```
數學表述：需要 P(x) = A 且 P(x) = ¬A（同一空間/時間/條件下不可能成立）
解法：找到一個維度（時間/空間/條件/尺度）使矛盾解耦
```

## 四大分離原則

### 1. 時間分離 (Separation in Time)

**核心問題**：屬性 A 和屬性 ¬A 是否需要在**不同時間**出現？

**物理本質**：時變系統 — P(t) 可以在 t₁ 時為 A，t₂ 時為 ¬A。任何有時間常數 τ 的物理過程都可利用（RC 電路 τ=RC、熱慣性 τ=ρcV/hA、化學反應 τ=1/k）。

| 策略 | 物理原理 | 跨域範例 |
|------|---------|---------|
| 預先動作 | 在 t < t_need 時預先完成，利用 stored energy E = ½kx² 或 E = ½CV² | 預應力混凝土（σ_residual 在澆築時施加，數十年後仍在工作）、超級電容預充電（ms 級放電需求用 s 級充電滿足） |
| 事後動作 | 利用 relaxation time τ_relax 在不需要後自動恢復 | 形狀記憶合金（A_f 溫度後恢復原形，Ni-Ti ε_rec ≈ 8%）、自癒合材料（裂紋觸發 Diels-Alder 逆反應→鍵合→癒合，τ ≈ hours） |
| 週期性切換 | 利用 f_switch >> f_disturbance 快速交替 | PWM 驅動（f=20kHz，負載看到等效直流）、脈衝雷射冷卻（on: 加工 10μs → off: 冷卻 990μs，duty 1%，峰值功率↑100×） |
| 加速通過 | 暴露劑量 D = P×t，t→0 使 D < D_threshold | 急冷淬火（通過 TTT nose 避免析出，v_cool > v_critical）、航太器快速穿越輻射帶（ΔD_radiation ∝ t_transit） |

**適用判斷**：若矛盾的兩個屬性在**不同操作階段**分別需要 → 時間分離。

---

### 2. 空間分離 (Separation in Space)

**核心問題**：屬性 A 和屬性 ¬A 是否可以在**不同位置**實現？

**物理本質**：空間梯度場 — P(x,y,z) 可以在位置 r₁ 為 A，位置 r₂ 為 ¬A。只要存在可控的空間梯度（∇P ≠ 0），就能實現空間分離。

| 策略 | 物理原理 | 跨域範例 |
|------|---------|---------|
| 局部品質/梯度材料 | 依 σ_vm(x,y,z) 分佈放置材料 | FGM 功能梯度材料（表面 ZrO₂ 隔熱 k=2 W/mK、基底 Ni 承載 σ_y=500 MPa）、人工牙根（表面多孔 HA 促骨整合、核心緻密鈦承載） |
| 分割/組合 | 界面設計使不同區域獨立 | 雙層管（內層耐蝕 316L、外層高強碳鋼）、三明治板（面板鋁承載、核心泡沫輕量） |
| 嵌套 | 同心結構 Z_inner ≠ Z_outer | 同軸電纜（內導體傳信號、外屏蔽隔 EMI）、雙壁真空杯（內壁反射、外壁隔熱） |
| 幾何變換 | 改變幾何使力/場在不同方向不同 | 碳纖維鋪層角度設計（0° 方向 E=140GPa 承載、90° 方向 E=10GPa 柔性）、非對稱翼型（上表面低壓=升力、下表面高壓=支撐） |

**適用判斷**：若矛盾的兩個屬性在**不同空間區域**分別需要 → 空間分離。

---

### 3. 條件分離 (Separation upon Condition)

**核心問題**：屬性 A 和屬性 ¬A 是否在**不同工作條件**下需要？

**物理本質**：非線性回應 — 材料/系統特性 P 是環境參數 X 的非線性函數 P(X)。在 X₁ 條件下 P=A，在 X₂ 條件下 P=¬A。關鍵是找到合適的控制參數 X（溫度、壓力、電場、磁場、頻率、pH...）。

| 策略 | 物理原理 | 跨域範例 |
|------|---------|---------|
| 相變/狀態變化 | Clausius-Clapeyron dP/dT = ΔH/(TΔV) — 相變時特性突變 | SMA 形狀記憶合金：T < M_f → 馬氏體（可變形）、T > A_f → 奧氏體（高彈性回復），NiTi 血管支架在體溫觸發展開；VO₂ 熱致變色玻璃：T < 68°C 透明→T > 68°C 反射紅外 |
| 參數閾值觸發 | 臨界現象 — 越過臨界點 X_c 時系統行為質變 | 限扭器（τ > τ_set → 打滑保護）、保險絲（I > I_rated → 熔斷 in ms）、超導體（T < T_c → ρ = 0，MRI 磁體 NbTi T_c=9.2K） |
| 環境響應材料 | 刺激回應聚合物 — ΔG_stimulus 驅動構型/體積變化 | pH 響應水凝膠（藥物緩釋，pH < 5 收縮釋放→pH 7 膨脹封閉）、光致變色鏡片（UV→分子異構化→深色）、自調節 PTC 加熱器（T↑→R↑→P↓→穩態） |
| 外場控制 | 施加/移除外場即時切換特性 | 磁流變液（B=0 液態→B≠0 固態，τ_y ∝ B²，響應 <1ms）、電致變色玻璃（V=0 透明→V≠0 著色）、電潤濕（接觸角 cos θ = cos θ₀ + εV²/2γd） |

**適用判斷**：若矛盾的兩個屬性在**不同操作條件**（溫度/載荷/頻率/環境）下分別需要 → 條件分離。

### 4. 整體與局部分離 (Separation between Whole and Parts)

**核心問題**：整體需要屬性 A，但局部可以用屬性 ¬A 組合出 A 的效果？

**物理本質**：湧現性 (Emergence) — 系統整體特性 ≠ 部件特性之和。透過巧妙排列 ¬A 的局部，可在系統層級實現 A（如：局部柔軟的鏈環 → 整體可承受巨大拉力）。

| 策略 | 物理原理 | 跨域範例 |
|------|---------|---------|
| 複合結構 | Rule of Mixtures: P_c = P₁V₁ + P₂V₂ — 兩種材料混合可得介於兩者之間的任意特性 | CFRP（碳纖維 σ=3500MPa + 環氧 ε_fail=5% → 高強高韌複合體）、鋼筋混凝土（鋼筋抗拉 + 混凝土抗壓 → 雙向承載） |
| 多孔/中空 | 相對密度 ρ*/ρ_s 控制 — Gibson-Ashby: σ*/σ_s ∝ (ρ*/ρ_s)^1.5，E*/E_s ∝ (ρ*/ρ_s)^2 | 蜂巢結構（ρ*/ρ_s ≈ 0.03，比剛度超過實心材料）、骨骼 trabecular 結構（Wolff's law 沿應力方向排列）、氣凝膠（ρ ≈ 3 kg/m³，k ≈ 0.013 W/mK） |
| 梯度/漸變 | 避免界面突變 → σ_interface → 0 | 梯度折射率透鏡 GRIN（n(r) 漸變，無球面像差）、梯度多孔鈦植入物（表面促骨生長、核心承載）、漸變齒形（噪音降低 5-10 dB，接觸應力均勻化） |
| 自相似/碎形 | 碎形維度 D > 拓撲維度，表面積/體積→∞ | 碎形天線（同一尺寸涵蓋多頻段，Sierpinski gasket 天線）、肺泡碎形結構（300M 個肺泡，表面積 70 m² 裝在 6L 體積內）、碎形散熱鰭片（有效散熱面積↑3-5×） |

**適用判斷**：若整體需要的宏觀特性可以由微觀層級的反向特性**組合/排列/湧現**出來 → 整體與局部分離。

## 分離原則選擇流程

```
物理矛盾: [物件] 需要同時 [屬性A] 和 [¬A]
                    │
        ┌───────────┼───────────┬──────────────┐
        ▼           ▼           ▼              ▼
   A和¬A在不同    A和¬A在不同  A和¬A在不同    整體要A
   時間出現？     位置需要？   條件下需要？   局部可以¬A？
        │           │           │              │
        ▼           ▼           ▼              ▼
   時間分離       空間分離     條件分離      整體局部分離
   (τ, f, dt)   (∇P ≠ 0)    (P = f(X))   (Emergence)
```

**第一性原理驗證**：選定分離策略後，必須能回答：
1. 控制方程是什麼？（哪個物理定律保證 A 和 ¬A 可以被分離？）
2. 分離的邊界條件是什麼？（在什麼範圍內此分離有效？）
3. 至少一個跨域實例證明此分離在物理上可行。

> **注意**：一個物理矛盾可能同時適用多個分離原則，應產出多條候選策略供後續篩選。
