## 一、物質-場（Su-Field）模型簡介

**Su-Field**（Substance-Field）模型是 TRIZ 中用於描述技術系統最小功能單元的建模方法：

- **S1（物質1）**：被作用的對象 — 注意其關鍵物性（密度、導熱率、楊氏模量、介電常數等）
- **S2（物質2）**：作用工具 — 注意其與 S1 的物理匹配性
- **F（場）**：S2 對 S1 施加作用所需的能量/場 — 注意其遵循的控制方程式

| 狀態 | 說明 | 對應物理現象 | 控制方程式舉例 |
|------|------|------------|--------------|
| **不完整系統** | 缺少 S1、S2 或 F | 能量傳遞路徑斷開 | 無法建立守恆方程 |
| **有效完整系統** | 三要素齊全且功能正常 | 能量從 S2 經 F 正確傳遞至 S1 | q = hAΔT (Newton cooling) |
| **有害完整系統** | 三要素齊全但產生有害效應 | 能量傳遞產生非預期耦合 | 如：熱傳路徑也傳導振動 |
| **不足效應系統** | 三要素齊全但作用效果不足 | 場的強度/密度/梯度不夠 | 如：B_gap < 0.3T → 扭矩不足 |

## Class 1：不改變或少量改變系統的改進

> 適用情境：系統結構基本不變，透過補全、消除有害作用或強化現有作用來改進。

### 1.1 建構 Su-Field 模型 — 2 個標準解

#### 1.1.1 建構完整 Su-Field（Build a Complete Su-Field）
若系統不完整（缺少 S1、S2 或 F），應補全缺失元素以形成完整的 Su-Field。

> **物理本質**：帕斯卡定律 — 密閉流體中壓力均勻傳遞（P = F/A）。**泛化**：任何需要遠端力傳遞且缺少傳遞介質的系統。**跨域**：達文西手術機器人用液壓場補全觸覺反饋 Su-Field；飛機操縱桿用液壓場傳遞舵面操控力。

#### 1.1.2 內部添加物（Add Internal Additive）
當無法直接建構完整 Su-Field 時，可在 S1 或 S2 內部添加物質。

> **物理本質**：Fourier 定律 — 複合材料等效熱導率 k_eff = Σ(k_i × φ_i)。添加高 k 填料提升 φ_high-k 分量。**泛化**：任何需要提升場傳遞效率但無法改變外部結構的系統。**跨域**：LED 封裝用氮化鋁填充矽膠（k 從 0.2→2.5 W/mK）[src: Razeeb et al. 2018 | paper]；IC 封裝用鑽石微粒導熱膏（k≈50 W/mK）[src: Element Six datasheet | datasheet]。

### 1.2 消除或中和有害作用 — 4 個標準解

#### 1.2.1 引入第三物質 S3（Introduce S3 to Eliminate Harmful Effect）
在 S1 與 S2 之間引入 S3 以消除或隔離有害作用。

> **物理本質**：振動阻抗不匹配 — Z = ρ × c，S3 的聲阻抗與 S1/S2 差異越大，能量反射率越高。**泛化**：任何「有害機械場傳遞」場景。**跨域**：半導體光刻機用空氣彈簧（Z ≈ 0）隔離地面微振，達 nm 精度。航太渦輪引擎用黏彈性阻尼環隔離葉片共振。

#### 1.2.2 引入 S1 或 S2 的變體（Introduce Modified S1 or S2）
用 S1 或 S2 自身的改良版本來消除有害效應。

> **物理本質**：材料阻尼比 ζ — 金屬 ζ ≈ 0.001，高分子 ζ ≈ 0.05-0.3。將 S2 從低阻尼材料換為高阻尼材料，能量在 S2 內部耗散而非傳遞至 S1。**跨域**：碳纖維複合材料齒輪（ζ ≈ 0.02）替代鋼齒輪（ζ ≈ 0.001），噪音降低 15-20 dB [src: Ashby Materials Selection | textbook]。船舶螺旋槳用銅鎳合金替代鋼（抑制空蝕振動）。

#### 1.2.3 引入反向場 F'（Introduce Counter-Field F'）
施加與有害場方向相反的場 F' 來中和。

> **物理本質**：疊加原理 — 兩個等幅反相波相加為零（Σ A_i = 0 when φ = π）。**泛化**：任何有害場可以被測量並反向複製的場景。**跨域**：Bose 主動消噪（聲場 F'）[src: Bose US Patent 4,455,675 | paper]、MRI 主動屏蔽線圈（磁場 F'，雜散場衰減 100×）[src: Mansfield & Chapman 1986 | paper]、電力系統有源濾波器（諧波電流 F'）。

#### 1.2.4 利用磁場與磁性物質（Use Magnetic Field + Ferromagnetic Substance）
利用磁場結合鐵磁物質隔離或消除有害效應。

> **物理本質**：磁通量守恆 — 高 μ 材料形成低磁阻路徑，雜散磁場被「吸入」屏蔽層而非穿透至被保護區（Φ = B × A = const）。**跨域**：電纜用 mu-metal 屏蔽（μ_r ≈ 50,000），CT 掃描室鐵板屏蔽，SQUID 超導量子干涉儀多層磁屏蔽。

### 1.3 進階合成與強化 — 7 個標準解

#### 1.3.1 鏈式 Su-Field（Chain Su-Field）
串聯多個 Su-Field，上一級輸出作為下一級輸入。

> **物理本質**：信號轉換鏈 — 每個 Su-Field 是一個 transducer，遵循能量守恆 η_total = Π(η_i)。**泛化**：任何需要將一種物理量多次轉換的系統。**跨域**：MEMS 加速度計鏈（慣性力→懸臂梁撓度→電容變化→ADC 數位信號）、光纖通訊（電信號→雷射光→光纖傳輸→光電二極體→電信號）。

#### 1.3.2 雙重 Su-Field（Double Su-Field）
疊加第二個場增強或細化作用效果。

> **物理本質**：能量疊加 — 兩個獨立場對 S1 做功 W_total = W_F1 + W_F2。**泛化**：任何單一場效果不足的場景。**跨域**：混合動力車（機械場+電磁場制動）、超音波輔助切削（機械場+超聲場降低切削力 30-50%）、微波+紅外線乾燥（穿透加熱+表面蒸發）。

#### 1.3.3 擴展至外部環境（Expand to External Environment）
將外部環境中可用的物質或場納入 Su-Field。

> **物理本質**：能量收割 — 環境中存在 free energy gradients（溫差 ΔT、壓差 ΔP、速度差 Δv），利用這些梯度做功。**跨域**：數據中心用外氣自然冷卻（air-side economizer）、風力渦輪用大氣壓梯度、熱電發電機利用工業廢熱（Seebeck effect，η ≈ 5-8%）。

#### 1.3.4 利用環境中的物質（Use Substances from External Environment）
直接利用環境中已有的物質作為 S3。

> **物理本質**：零成本資源利用 — 環境中已存在的物質無需採購或運輸（化學勢 μ_env = 0）。**跨域**：海水淡化利用海水本身作為冷卻劑、沙漠建築用砂石作為蓄熱介質（比熱 c ≈ 0.8 kJ/kgK）、生物反應器用空氣中的 O₂ 作為氧化劑。

#### 1.3.5 微量添加物（Use Minimal Additive）
引入微量物質即可達到改善效果。

> **物理本質**：Stribeck 曲線 — 邊界潤滑區間，奈米粒子填入表面微凸體谷底，真實接觸面積↓，摩擦係數從 μ ≈ 0.3 → 0.05。**泛化**：任何界面現象可被微量物質改變的場景。**跨域**：鋼鐵冶煉加微量 V/Nb（0.01%）提升晶粒細化強度 30%（Hall-Petch 方程 σ_y ∝ d^(-1/2)）；蛋白質藥物加微量表面活性劑防止聚集（降低界面張力 γ）。

#### 1.3.6 大量添加後移除（Use Maximal Additive then Remove）
暫時引入大量物質完成功能，之後移除。

> **物理本質**：犧牲性介質 — 暫時改變邊界條件完成加工/保護，再恢復原邊界條件。**跨域**：鑄造用砂模（成型後敲碎移除）、手術中冰凍切片（冰作為臨時支撐，切完融化）、半導體蝕刻用光阻（曝光顯影後剝離）。本質是 F(t) = F_active (0 < t < t_end) → 0。

#### 1.3.7 場替代物質（Use Field Instead of Substance）
用場達到原本需要物質才能達到的功能。

> **物理本質**：非接觸力 — 場（電、磁、聲、光）可在無物質接觸下傳遞能量或資訊（F = qE, F = qv×B, P = I²/2ρc）。**跨域**：超音波清洗替代化學溶劑、雷射焊接替代助焊劑、靜電噴塗替代浸泡塗裝（減少材料浪費 40%）。

## Class 2：改變系統的改進

> 適用情境：現有系統結構不足以滿足需求，需要進行結構性變更。

### 2.1 轉換為雙系統或多系統 — 6 個標準解

#### 2.1.1 建立雙系統（Create Bi- or Poly-System）
將兩個或多個相同系統並聯以增強功能。

> **物理本質**：並聯定理 — 相同系統並聯，輸出能力加法（P_total = ΣP_i），可靠度乘法（R_sys = 1 - Π(1-R_i)）。**跨域**：RAID 磁碟陣列（容量↑+冗餘）、多引擎飛機（推力↑+安全）、並聯電池組（容量↑）。

#### 2.1.2 強化雙系統（Create Enhanced Bi-System）
兩個互補子系統結合，彌補彼此不足。

> **物理本質**：Ragone 圖互補 — 高能量密度（電池）與高功率密度（超級電容）在 E-P 空間互補。**泛化**：任何兩個系統在不同物性維度各有優勢。**跨域**：F1 賽車 KERS（動能回收+渦輪增壓）、混合記憶體（DRAM 速度 + NAND 容量）、雙目視覺（左眼解析度 + 右眼深度）。

#### 2.1.3 反向雙系統（Create Inverse Bi-System）
兩個性質相反的子系統組合。

> **物理本質**：Peltier 效應 — 同一元件施加正/反向電流可加熱或冷卻（Q = π × I，π 為 Peltier 係數）。**泛化**：任何需要雙向操控的系統。**跨域**：熱電元件（加熱+冷卻）、H 橋馬達驅動（正轉+反轉）、壓電致動器（伸+縮）。

#### 2.1.4 組合不相容特性（Combine Incompatible Properties）
在同一系統中實現看似矛盾的特性。

> **物理本質**：各向異性 — 複合材料纖維方向決定楊氏模量 E(θ)，不同方向可獨立調控力學特性。**泛化**：任何需要「同一部件不同區域不同物性」。**跨域**：飛機機翼蒙皮（翼根 E=120 GPa 承載、翼尖 E=40 GPa 氣彈減振）、人工膝關節梯度多孔鈦（表面多孔促骨整合、核心緻密承載）。

#### 2.1.5 增加元素差異性（Increase Difference Between Elements）
增大多系統中各元素的差異以產生新功能。

> **物理本質**：頻率分離 — 不同頻率響應的元件組合擴大系統頻寬（BW_sys > max(BW_i)）。**跨域**：多頻帶天線（不同長度振子覆蓋不同頻段）、分光鏡組（不同介質層分離不同波長）、寬頻壓電能量收集（不同尺寸懸臂梁覆蓋 10-1000 Hz）。

#### 2.1.6 多系統收斂為單系統（Fold Poly-System into Mono-System）
功能整合至單一元件。

> **物理本質**：功能密度最大化 — 將 N 個獨立功能整合至一個物理實體，減少界面損耗（Σ R_interface → 0）。**跨域**：SoC 晶片（CPU+GPU+Memory+IO 整合）、智慧手錶（通訊+感測+顯示+計算）、MEMS 慣性測量單元（3 軸加速度+3 軸陀螺=1 顆晶片）。

### 2.2 發展 Su-Field — 6 個標準解

#### 2.2.1 以更可控場替代（Replace with More Controllable Field）
可控性排序：機械 < 熱 < 化學 < 電 < 磁。

> **物理本質**：控制頻寬 — 電磁場的控制迴路頻寬（kHz-MHz）遠高於機械場（Hz-kHz）。**跨域**：磁流變阻尼器替代液壓閥（響應 1 ms vs 50 ms）[src: Carlson 2005 | paper]、電子束焊替代火焰焊（聚焦精度 μm vs mm）、壓電閥替代電磁閥（kHz vs 100 Hz）。

#### 2.2.2 碎片化 S2（Fragment S2）
將 S2 分解為更細小的顆粒或碎片。

> **物理本質**：Newton 冷卻定律 q = hA(T_s-T_∞)，碎片化使表面積 A 增大 5-10×，對流係數 h 因湍流增強也提升。**跨域**：微通道散熱器（A/V 比高 100×）、催化轉換器（貴金屬奈米粒子 vs 塊材，活性面積↑10,000×）、噴霧乾燥（液滴碎片化加速蒸發）。

#### 2.2.3 毛細管/多孔結構（Use Capillary / Porous Structures）
引入毛細管或多孔材料增強物質傳輸。

> **物理本質**：Young-Laplace 方程 ΔP = 2γcosθ/r — 小孔徑 r 產生大毛細壓力驅動液體。**跨域**：熱管蒸發器（毛細芯驅動工質循環，等效 k > 10,000 W/mK）[src: Faghri 2016 Heat Pipe Science | textbook]、紙基微流道（μPAD 醫療快篩）[src: Martinez et al. 2007 | paper]、樹木導管（木質素毛細管輸送水分至 100m 高）[src: Tyree 1997 | paper]。

#### 2.2.4 增加動態性（Increase Dynamism）
使系統從剛性轉為可變形/可調節。

> **物理本質**：自由度增加 — 固定 DOF=0 → 可調 DOF≥1，系統可適應時變邊界條件。**跨域**：可變幾何渦輪（葉片角度隨轉速調整）、自適應光學望遠鏡（可變形鏡面補償大氣擾動）、軟體機器人（連續體結構 DOF→∞）。

#### 2.2.5 結構化場（Structure the Field）
將均勻場改為非均勻場。

> **物理本質**：梯度場 — 非均勻場在不同空間位置施加不同強度，實現選擇性作用（F = ∇(m·B) for magnetic gradient）。**跨域**：MRI 梯度線圈（空間定位）、聚焦超聲手術刀（HIFU，焦點溫度 >60°C 而周圍 <42°C）、離子佈植（能量梯度控制摻雜深度）。

#### 2.2.6 結構化物質（Structure the Substance）
將均質物質改為非均質結構。

> **物理本質**：應力最佳化 — 依 von Mises 應力分佈放置材料，σ_max/σ_avg → 1（Michell truss 理論）。**跨域**：骨骼 trabecular 結構（沿應力軌跡排列，Wolff's law）、拓撲最佳化 3D 列印支架（移除低應力區域減重 40-60%）、變壁厚壓力容器。

### 2.3 節奏與共振 — 3 個標準解

#### 2.3.1 場的節奏匹配（Match Field Rhythm to System Rhythm）

> **物理本質**：共振/反共振 — 當 f_external = f_natural 時振幅→∞（共振，利用）或刻意避開（反共振，防害）。**跨域**：超聲波清洗匹配液體空化頻率、建築物 TMD（調諧質量阻尼器）匹配樓層自然頻率、核磁共振 RF 脈衝匹配 Larmor 頻率。

#### 2.3.2 兩場節奏匹配（Match Rhythms of Two Fields）

> **物理本質**：同步 — 兩個場的頻率/相位鎖定可最大化能量轉換效率（如 η_max when f₁ = f₂ and Δφ = 0）。**跨域**：鎖相環 PLL（相位同步）、同步整流（MOSFET 開關與電流相位匹配）、心臟節律器雙腔同步（心房-心室電場同步）。

#### 2.3.3 利用不同場的交互作用（Use Two Incompatible Fields）

> **物理本質**：拍頻效應 — 兩個頻率接近的場疊加產生調幅波（f_beat = |f₁ - f₂|），可用於感測或混頻。**跨域**：外差式光學干涉儀（nm 位移量測）、雷達脈衝壓縮（頻率掃描 + 匹配濾波）、原子力顯微鏡 tapping mode（機械+壓電場交互）。

---

### 2.4 鐵磁物質與磁場 — 8 個標準解

#### 2.4.1 引入鐵磁物質

> **物理本質**：磁化率 χ — 鐵磁材料 χ ≈ 10³-10⁵，在外部磁場中產生強烈力學回應（F = χVB∇B/μ₀）。**跨域**：磁流變阻尼器（可變阻尼 10×範圍）、磁性奈米粒子標靶藥物遞送、磁選礦（分離不同磁化率的礦物）。

#### 2.4.2 利用鐵磁顆粒

> **物理本質**：顆粒群體行為 — 單域磁粒（D < D_critical ≈ 30nm for Fe₃O₄）具有超順磁性，無磁滯損耗。**跨域**：MRI 對比劑（超順磁氧化鐵 SPIO）、磁性流體軸承密封（零摩擦零洩漏）、磁珠免疫分析（快速分離生物標記物）。

#### 2.4.3 外部磁場控制（Use External Magnetic Field）

> **物理本質**：Lorentz 力 F = qv×B — 外部磁場可遠端、非接觸、即時操控帶磁/帶電物體。**跨域**：磁懸浮列車（Meissner effect 或電磁懸浮）、電弧焊磁偏吹控制、粒子加速器束流導引。

#### 2.4.4 磁流體（Use Ferrofluid）

#### 2.4.5 複合鐵磁結構

## Class 4：檢測與測量

> 適用情境：需要檢測或測量某個參數但現有方法不可行或不夠精確。

### 4.1 間接方法 — 3 個標準解

#### 4.1.1 以模型替代直接測量

> **物理本質**：物理量耦合 — 不可直接觀測的量 X 與可觀測量 Y 有確定的物理關係 X = f(Y)。**跨域**：電池 SOC 由 OCV 推算（Nernst 方程）、建築結構應力由應變片推算（Hooke's law σ = Eε）、腫瘤溫度由 MRI T1 弛緩時間推算。

#### 4.1.2 使用副本測量（Use Copies for Measurement）
若無法直接測量原物件，可對其副本或複製品進行測量。

> **物理本質**：幾何相似律 — 副本與原物件保持尺度因子 λ 的映射關係，物理量可透過相似準則換算（如 Re 數相似保證流場一致）。**跨域**：風洞模型（1:10 縮尺模型+Re 數匹配測氣動力）、地震模擬振動台（結構縮尺模型驗證抗震）、數位孿生（Digital Twin 用虛擬副本預測真實系統狀態）。

#### 4.1.3 連續兩次測量（Use Two Successive Detections）
透過兩次連續測量的差值來檢測變化量。

> **物理本質**：差分法 — ΔX = X(t₂) - X(t₁)，消除系統偏差和共模干擾，只保留變化量（SNR↑）。**跨域**：干涉測量（雙光束干涉消除背景光→nm 位移量測）、差分 GPS（兩站接收消除大氣誤差→cm 精度）、差示掃描量熱 DSC（雙坩堝差分測量→偵測 μW 級相變熱）。

### 4.2 建構測量 Su-Field — 4 個標準解

#### 4.2.1 建構測量用 Su-Field

> **物理本質**：感測器三要素 — 被測物（S1）+ 感測元件（S2）+ 探測場（F），遵循特定 transduction 方程。**跨域**：Hall effect 轉速量測（F=磁場，V_H = I×B/ned）、超聲波流量計（F=聲波，v = Δt×c²/2L）、光學編碼器（F=光場，光柵干涉）。

#### 4.2.2 內部添加物輔助測量（Add Internal Additive to S1 for Detection）
在被測物 S1 中添加易於偵測的物質，使其狀態變化可被間接觀測。

> **物理本質**：示蹤劑動力學 — 添加物濃度 C(t) 隨系統狀態變化，追蹤 C(t) 等效追蹤目標參數（如 τ = V/Q 停留時間分佈）。**跨域**：螢光示蹤劑檢測管路洩漏（UV 激發→可見光發射，偵測限 ppb 級）、放射性同位素標記追蹤藥物代謝路徑（PET 掃描）、煙霧測試偵測建築氣密性（可視化氣流路徑）。

#### 4.2.3 外部環境添加物輔助測量（Add Additive to External Environment）
若無法在 S1 內部添加物質，改在外部環境中添加。

> **物理本質**：散射/吸收對比 — 環境介質的光學/聲學/電磁特性改變後，S1 與背景的對比度 ΔI/I_bg 增大。**跨域**：超聲波耦合劑（消除空氣界面反射→成像清晰度↑）、CT 對比劑（碘劑改變組織 X 射線吸收係數 Δμ）、氣泡流場可視化（PIV 粒子散射光追蹤流速場）。

#### 4.2.4 從環境獲取測量資訊（Obtain Measurement Data from Environment）
不在系統中添加任何物質，而是利用環境中已有的場或物質獲取測量資訊。

> **物理本質**：被動感測 — 利用環境中天然存在的信號源（熱輻射、地磁場、環境振動），被測物對這些信號的調制攜帶狀態資訊。**跨域**：紅外線熱成像（被動偵測物體自身熱輻射 W = εσT⁴）、被動聲納（偵測目標自身噪音而非發射探測波）、環境振動模態分析（利用風/交通微振動取代人工激振，operational modal analysis）。

### 4.3 強化測量 Su-Field — 3 個標準解

#### 4.3.1 強化現有測量 Su-Field

> **物理本質**：信噪比增強 — 加入放大場或共振機制使 SNR = S/N 提升。**跨域**：鎖相放大器（窄帶濾波 SNR↑1000×）、磁致伸縮塗層放大應變信號、SERS 表面增強拉曼散射（奈米金粒子放大拉曼信號 10⁶×）。

#### 4.3.2 鏈式測量 Su-Field（Chain Measurement Su-Field）
將測量系統串聯為多級放大鏈，逐級提升信號強度或轉換為更易偵測的形式。

> **物理本質**：級聯增益 — 多級轉換器串聯，總增益 G_total = Π(G_i)，微弱信號經多次放大達到可量測範圍。**跨域**：光電倍增管 PMT（光子→光電子→二次電子級聯放大 10⁶×）、儀表放大器三級架構（差模放大+共模抑制→CMRR > 120 dB）、掃描隧道顯微鏡 STM（隧穿電流 nA→前置放大→回饋控制→原子級影像）。

#### 4.3.3 雙重測量 Su-Field（Double Measurement Su-Field）
疊加第二個場來增強測量敏感度或提供額外的測量維度。

> **物理本質**：多模態融合 — 兩種獨立的場提供互補資訊，聯合估計精度優於單場（Fisher 資訊量加法 I_total = I_F1 + I_F2）。**跨域**：PET-CT（代謝活性 + 解剖結構雙模態融合）、電磁超聲 EMAT（電磁場激發 + 超聲場接收，非接觸式缺陷檢測）、紅外+可見光雙波段監控（熱特徵+紋理特徵→全天候目標識別）。

### 4.4 鐵磁測量 Su-Field — 2 個標準解

#### 4.4.1 鐵磁標記物測量

> **物理本質**：磁標記追蹤 — 鐵磁標記的磁矩 m 可被外部磁感測器遠端偵測，靈敏度 ~nT（SQUID）。**跨域**：手術器械追蹤（磁標記+外部陣列）、管線 PIG 檢測（磁漏檢查腐蝕）、生物標記磁免疫分析（magnetic ELISA）。

#### 4.4.2 鐵磁 Su-Field 複合測量（Complex Ferromagnetic Measurement Su-Field）
利用鐵磁材料的磁特性變化作為被測物理量的指標。

> **物理本質**：磁彈效應（Villari effect）— 鐵磁材料在應力作用下磁導率改變 Δμ/μ = f(σ)，磁通量變化可被偵測。逆磁致伸縮方程 ΔB = d* × σ。**跨域**：磁彈性扭矩感測器（方向盤 EPS 扭矩量測，精度 ±0.5%）、Barkhausen 噪聲檢測殘餘應力（磁疇壁運動→感應脈衝統計分析）、磁記憶法金屬疲勞早期診斷（應力集中區自發磁化異常）。

### 4.5 測量系統演化方向 — 5 個標準解

#### 4.5.1 利用物理/化學效應測量

> **物理本質**：能量轉換型感測 — 利用壓電效應（δ = d × σ）、熱電效應（V = α × ΔT）、光電效應（I = η × Φ × e/hν）等進行直接轉換。**跨域**：壓電薄膜輪胎壓力感測、光纖光柵溫度/應變感測（Bragg 波長漂移 Δλ = 2nΛ(α+ξ)ΔT）。

#### 4.5.2 利用共振測量

> **物理本質**：共振頻率位移 — 系統質量 m 或剛度 k 改變時，f_n = (1/2π)√(k/m) 偏移，偵測 Δf 即可推算 Δm 或 Δk。**跨域**：石英微天平 QCM（質量靈敏度 ~ng/cm²）、原子力顯微鏡（探針共振頻率→表面力）、橋梁健康監測（自然頻率偏移→結構劣化）。

#### 4.5.3 雙系統或多系統測量（Bi-System / Poly-System Measurement）
將單一測量系統擴展為雙系統或多系統，利用差分或冗餘來提升精度。

> **物理本質**：差分消除 — 兩個結構相同的感測器受相同干擾但信號極性相反，相減後 S_diff = 2S_signal, N_diff ≈ 0（共模抑制）。**跨域**：惠斯通電橋應變量測（四臂差分消除溫度漂移）、差動變壓器 LVDT（雙線圈差分→位移線性度 ±0.1%）、雙光路干涉儀（參考光路消除環境振動）。

#### 4.5.4 測量導數而非絕對值（Measure Derivative Instead of Absolute Value）
當絕對值難以測量時，改測其變化率（導數）或積分值。

> **物理本質**：微分/積分變換 — Faraday 定律 EMF = -dΦ/dt 即典型的「測導數」：不直接測磁通量 Φ，而是測其時間變化率。反之，陀螺儀積分角速度得角度 θ = ∫ω dt。**跨域**：加速度計積分得速度/位移（慣性導航）、流量計微分得加速度（水錘監測 dP/dt）、渦電流探傷（測阻抗變化率而非絕對阻抗→靈敏度↑10×）。

#### 4.5.5 預先偵測（Prior Detection / Anticipatory Measurement）
在事件發生前預先佈置測量系統，捕捉初始瞬態信號。

> **物理本質**：前驅信號捕獲 — 許多失效模式在巨觀破壞前有微觀前驅信號（聲發射 AE 能量 E_AE ∝ (σ/σ_f)^m，m ≈ 4-8，破壞前 AE 事件指數增長）。**跨域**：聲發射監測壓力容器疲勞裂紋（Kaiser effect 判斷歷史最大應力）、地震 P 波預警系統（P 波速 6 km/s > S 波 3.5 km/s，爭取數秒預警）、鋰電池熱失控預測（偵測內阻異常上升 ΔR/R > 5% 觸發預警）。

### 5.3 相變與物態轉換 — 5 個標準解

#### 5.3.1 利用相變（Use Phase Transition）

> **物理本質**：潛熱儲存 — 相變過程溫度不變但吸收/釋放大量熱（ΔH_fus）。**泛化**：任何需要「恆溫緩衝」的系統。**跨域**：衛星熱控石蠟 PCM（ΔH≈200 kJ/kg）、建築 PCM 牆體降低空調負荷 20-30%、低溫手術冰漿灌注、數據中心 PCM 備用冷卻。

#### 5.3.2 雙態物質（Use Dual-State Substance）

> **物理本質**：馬氏體相變 — 無擴散剪切變形，A_f（奧氏體完成溫度）決定切換點，可逆應變 ε_rec ≈ 6-8%。**跨域**：NiTi 形狀記憶合金血管支架（體溫觸發展開）、衛星太陽能板鉸鏈（太空冷→展開→加熱鎖定）、火災灑水頭（熱敏合金觸發）。

#### 5.3.3 相變伴隨效應

> **物理本質**：體積變化/潛熱/表面張力變化 — 如水結冰 ΔV/V ≈ +9%，可產生 GPa 等級壓力。**跨域**：凍裂膨脹破碎岩石（採礦）、凍乾保存生物製劑（昇華 ice→vapor）、氣泡核沸騰增強傳熱（ΔT_sat 驅動成核密度）。

#### 5.3.4 兩相物質替代單相

> **物理本質**：相變傳熱 — 兩相流的等效熱導率 k_eff 可達 10,000-100,000 W/mK（熱管原理，蒸發→冷凝循環）。**跨域**：熱管散熱（筆電→衛星→核反應器）、蒸發冷卻（冷卻塔、人體排汗）、兩相流噴射冷卻（伺服器晶片散熱密度 >100 W/cm²）。

#### 5.3.5 相互作用的物質（Physicochemical Effects）

> **物理本質**：化學反應動力學 — 利用反應的能量釋放/吸收或產物特性實現功能（ΔG < 0 自發反應）。**跨域**：自修復材料（微膠囊破裂→環氧固化→裂紋癒合，White et al. 2001）、安全氣囊（NaN₃ 分解→N₂ 氣體）、化學發光棒（H₂O₂ + 螢光劑→光子）。

### 5.4 利用物理效應 — 2 個標準解

#### 5.4.1 自適應物質

> **物理本質**：回饋迴路內建於材料 — 材料特性 P 是環境變數 X 的函數 P = f(X)，無需外部控制器。**跨域**：光致變色鏡片（UV→分子構型變化→吸收光譜改變）、自調節加熱器（PTC 熱敏電阻，T↑→R↑→P↓）、pH 響應水凝膠藥物緩釋。

#### 5.4.2 多物理效應耦合

> **物理本質**：交叉效應矩陣 — 壓電（機械↔電）、磁致伸縮（磁↔機械）、熱電（熱↔電）、光聲（光↔聲）等耦合效應可實現跨域能量轉換。**跨域**：壓電-電磁複合能量收集、光聲成像（雷射脈衝→熱膨脹→超聲波→3D 影像）、多鐵材料（同時具有鐵電+鐵磁性→電控磁性）。

### 5.5 實驗方法獲得微粒子 — 3 個標準解

#### 5.5.1 分解或合成獲得所需物質

> **物理本質**：原位合成 — 在需要的位置和時間產生所需物質（化學平衡 K = exp(-ΔG/RT)）。**跨域**：電解水原位產氫（燃料電池）、CVD 原位沉積薄膜（半導體）、3D 生物列印原位聚合水凝膠支架。

#### 5.5.2 從系統元素分解獲得

> **物理本質**：副產物利用 — 系統運作過程中的副產物（廢熱、副反應氣體、磨耗粒子）可被轉化為有用資源。**跨域**：高爐煤氣回收發電、氫燃料電池副產物水用於加濕膜、CMP 研磨漿回收再利用。

#### 5.5.3 添加物反應產物

> **物理本質**：設計反應路徑 — 選擇添加物使其反應產物具有目標功能（reactant A + additive B → product C + functional byproduct D）。**跨域**：免清洗助焊劑（殘留物即防腐層）、自蝕刻犧牲層（反應產物為保護膜）、水泥自癒合（細菌分泌�ite 填充裂縫，生物化學路徑）。

## 三、標準解匹配流程（Quick Reference）

```
1. 建立 Su-Field 模型（命名 S1, S2, F 及其遵循的物理定律）
   ├─ 不完整系統 → Class 1.1（建構 Su-Field）
   ├─ 有害效應系統 → Class 1.2（消除有害作用）
   ├─ 不足效應系統 → Class 1.3 / Class 2（強化或改變系統）
   └─ 測量問題 → Class 4（檢測與測量）

2. 系統改變方向
   ├─ 結構不變 → Class 1
   ├─ 結構改變 → Class 2
   ├─ 層級改變 → Class 3
   └─ 實施簡化 → Class 5

3. 第一性原理驗證
   ├─ 每個候選解必須追溯到一個控制方程式
   ├─ 至少一個跨域實例證明該物理機制已被驗證
   └─ 對比候選解，選擇最符合「理想性」且有最多 E2+ 證據的方案
```
